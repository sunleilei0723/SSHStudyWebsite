#240 搜索二维矩阵II

编写一个高效的算法来搜索 m × n 矩阵中的一个目标值 target。该矩阵具有以下特性：每行的元素从左到右升序排列，每列的元素从上到下升序排列。

示例 1
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5
输出：true
示例 2
输入：target = 20
输出：false
2 题目拆解
行列有序矩阵 ➜ 搜索target ➜ 从右上角出发 ➜ 找到/排除

关键约束：

  • 每行从左到右升序，每列从上到下升序
  • 不同于 74 题，本题行首不一定大于上一行尾
  • 要求高效算法 → 不能逐行二分 O(mlogn)，要 O(m+n)
3 解题思路

右上角搜索法 O(m+n)

从右上角出发，当前位置 matrix[row][col]：如果等于 target 找到；如果大于 target，说明该列下面的更大，左移 col--；如果小于 target，说明该行左边更小，下移 row++。每次排除一行或一列。

右上角开始 ➜ 当前值 > target → 左移 ➜ 当前值 < target → 下移 ➜ 等于 → 找到！
4 代码演示
def searchMatrix(matrix, target): if not matrix or not matrix[0]: return False m, n = len(matrix), len(matrix[0]) row, col = 0, n - 1 # 从右上角出发 while row < m and col >= 0: if matrix[row][col] == target: return True elif matrix[row][col] > target: col -= 1 # 当前值太大 → 左移 else: row += 1 # 当前值太小 → 下移 return False
public boolean searchMatrix(int[][] matrix, int target) { int m = matrix.length, n = matrix[0].length; int row = 0, col = n - 1; // 从右上角出发 while (row < m && col >= 0) { if (matrix[row][col] == target) return true; else if (matrix[row][col] > target) col--; else row++; } return false; }

6 易错要点
起点选择

只能从右上角或左下角出发。左上角和右下角无法决定移动方向（同侧元素大小关系不确定）。

与74题区别

74 题行首严格大于上一行尾（可视为一维有序），可用二分。本题仅行列有序，不能直接二分。

边界条件

空矩阵 [] 或 [[]] 需单独处理，否则访问 matrix[0] 会越界。

7 扩展思考
  • 左下角出发：同理，当前值 > target 上移，< target 右移，效果对称
  • 逐行二分：对每行做二分查找，O(mlogn)，当 m 远小于 n 时可接受
  • 四分剪枝：从中间开始，递归排除不可能的象限，最坏 O((mn)^log₄3)
  • 如果行列严格有序：即 74 题场景，可先定位行再二分，O(log(mn))
