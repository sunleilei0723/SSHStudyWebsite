#121 买卖股票的最佳时机

给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。你只能选择某一天买入并在未来某一天卖出，设计算法计算最大利润。如果不能获得利润，返回 0。

示例 1
输入：prices = [7,1,5,3,6,4]
输出：5
第 2 天买入（价格 = 1），第 5 天卖出（价格 = 6），利润 = 6-1 = 5
示例 2
输入：prices = [7,6,4,3,1]
输出：0
价格持续下降，无法获得利润
2 题目拆解
价格序列 ➜ 选一天买入 ➜ 选后面一天卖出 ➜ 最大利润

关键约束：

  • 只能交易一次：一次买入 + 一次卖出
  • 买入必须在卖出之前（先买后卖）
  • 没有利润时返回 0（可以不交易）
3 解题思路

暴力法 O(n²)：枚举所有买入卖出日 → 太慢

贪心法 O(n)：遍历时维护历史最低价，每天计算"今天卖出的利润"，取最大值：

遍历每一天 ➜ 更新最低价 ➜ 计算当天利润 ➜ 更新最大利润
4 代码演示
def maxProfit(prices): min_price = float('inf') max_profit = 0 for price in prices: min_price = min(min_price, price) # 更新最低价 max_profit = max(max_profit, price - min_price) # 更新最大利润 return max_profit
public int maxProfit(int[] prices) { int minPrice = Integer.MAX_VALUE; int maxProfit = 0; for (int price : prices) { minPrice = Math.min(minPrice, price); maxProfit = Math.max(maxProfit, price - minPrice); } return maxProfit; }
int maxProfit(vector<int>& prices) { int minPrice = INT_MAX; int maxProfit = 0; for (int price : prices) { minPrice = min(minPrice, price); maxProfit = max(maxProfit, price - minPrice); } return maxProfit; }

6 易错要点
先更新最低价再算利润

必须先 min_price = min(min_price, price) 再算利润，否则当天买入当天卖出利润为 0，但题目不允许同天买卖。

初始值设定

min_price 初始化为 ∞ 而非 prices[0]，max_profit 初始化为 0 而非负数，因为最差情况就是不交易。

只能交易一次

本题限制只能买卖一次。如果允许多次交易是 #122，如果最多两次是 #123，注意区分。
