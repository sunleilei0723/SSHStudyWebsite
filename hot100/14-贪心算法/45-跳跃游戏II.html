#45 跳跃游戏II

给定一个长度为 n 的 0 索引整数数组 nums，初始位置在 nums[0]。每个元素表示从该位置可以跳跃的最大长度。你的目标是使用最少的跳跃次数到达数组的最后一个位置。假设总是可以到达最后一个位置。

示例 1
输入：nums = [2,3,1,1,4]
输出：2
从下标 0 跳到下标 1（跳 1 步），再从下标 1 跳到下标 4（跳 3 步），共 2 步
示例 2
输入：nums = [2,3,0,1,4]
输出：2
2 题目拆解
起点 index=0 ➜ 每格可跳 nums[i] 步 ➜ 最少跳跃次数

关键约束：

  • 题目保证一定可以到达终点
  • 求的是最少步数，不是能否到达
  • 每个位置可以跳 1 到 nums[i] 步
3 解题思路

贪心 + BFS 思想 O(n)：把跳跃过程看作逐层扩展，每一步选择能到达最远的位置：

当前层边界 end ➜ 遍历层内找最远 ➜ 到达边界→步数+1 ➜ 更新边界 end

维护 end（当前步数能到的最远边界）和 maxReach（下一步能到的最远位置）。当 i == end 时，步数 +1，end = maxReach。

4 代码演示
def jump(nums): steps = end = max_reach = 0 for i in range(len(nums) - 1): max_reach = max(max_reach, i + nums[i]) # 更新最远可达 if i == end: # 到达当前步的边界 steps += 1 # 必须再跳一步 end = max_reach # 新边界 return steps
public int jump(int[] nums) { int steps = 0, end = 0, maxReach = 0; for (int i = 0; i < nums.length - 1; i++) { maxReach = Math.max(maxReach, i + nums[i]); if (i == end) { steps++; end = maxReach; } } return steps; }
int jump(vector<int>& nums) { int steps = 0, end = 0, maxReach = 0; for (int i = 0; i < nums.size() - 1; i++) { maxReach = max(maxReach, i + nums[i]); if (i == end) { steps++; end = maxReach; } } return steps; }

6 易错要点
遍历到 n-2

循环只需到 len(nums)-2，因为到达最后一个位置时不需要再跳了。遍历到 n-1 会多算一步。

先更新 maxReach 再判断

必须先 maxReach = max(maxReach, i+nums[i])，再判断 i==end。确保在边界处已收集完所有可达信息。

不是选最大 nums[i]

贪心选的是 i + nums[i] 最大的位置，不是 nums[i] 最大的。位置靠后 + 跳跃力强 = 最远可达。

7 扩展思考
  • BFS 视角：本题本质是 BFS 层序遍历，每一步是一层，步数就是层数
  • DP 解法 O(n²)：dp[i] = 到达 i 的最少步数，但不如贪心高效
  • 跳跃游戏 I (#55)：只需判断能否到达，维护 maxReach 即可
  • 如果要求路径：在贪心基础上记录每步的起跳位置即可还原路径
