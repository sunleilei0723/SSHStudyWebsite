#763 划分字母区间

给定一个字符串 s ，将该字符串划分为尽可能多的片段，使得同一字母最多出现在一个片段中。返回一个表示每个字符串片段长度的列表。

示例 1
输入：s = "ababcbacadefegdehijhklij"
输出：[9,7,8]
分区为 "ababcbaca"、"defegde"、"hijhklij"，每个字母只出现在一个分区中
示例 2
输入：s = "eccbbbbdec"
输出：[10]
2 题目拆解
字符串 ➜ 同一字母同一段 ➜ 尽可能多分段 ➜ 每段长度

关键约束：

  • 同一字母只能出现在一个片段中
  • 要尽可能多的分段（贪心：尽早切割）
  • 返回的是每段的长度，不是子串
3 解题思路

贪心法 O(n)：先统计每个字母的最后出现位置，再遍历扩展区间：

统计 last[char] ➜ 遍历扩展 end ➜ i == end → 切割 ➜ 记录长度

遍历时 end = max(end, last[s[i]])，当 i == end 时说明当前区间内所有字母的最后位置都已包含，可以切割。

4 代码演示
def partitionLabels(s): last = {c: i for i, c in enumerate(s)} # 每个字母最后位置 result = [] start = end = 0 for i, c in enumerate(s): end = max(end, last[c]) # 扩展区间右边界 if i == end: # 所有字母最后位置都已包含 result.append(end - start + 1) # 记录长度 start = i + 1 # 新区间起点 return result
public List<Integer> partitionLabels(String s) { int[] last = new int[26]; for (int i = 0; i < s.length(); i++) last[s.charAt(i) - 'a'] = i; List<Integer> res = new ArrayList<>(); int start = 0, end = 0; for (int i = 0; i < s.length(); i++) { end = Math.max(end, last[s.charAt(i) - 'a']); if (i == end) { res.add(end - start + 1); start = i + 1; } } return res; }

6 易错要点
先统计再遍历

必须先完整统计每个字母的最后出现位置，再遍历扩展区间。不能边遍历边统计，否则会遗漏后面的位置。

end 是动态扩展的

end 不是固定的，每遇到新字母都可能扩展。只有当 i == end 时才确定可以切割。

长度计算

片段长度 = end - start + 1，别忘了 +1。切割后 start = i + 1，不是 end + 1。

7 扩展思考
  • 合并区间 (#56)：本题是合并区间的变体，先求每个字母的区间 [first, last]，再合并
  • 如果要求最少分段：那就是合并区间问题，把重叠区间合并
  • 如果字母范围更大：用 HashMap 代替数组，逻辑不变
