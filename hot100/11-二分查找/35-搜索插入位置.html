#35 搜索插入位置

给定一个排序数组 nums 和目标值 target，在数组中找到目标值并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。请必须使用时间复杂度为 O(log n) 的算法。

示例 1
输入：nums = [1,3,5,6], target = 5
输出：2
示例 2
输入：nums = [1,3,5,6], target = 2
输出：1
示例 3
输入：nums = [1,3,5,6], target = 7
输出：4
2 题目拆解
排序数组 ➜ 目标值 target ➜ 二分查找位置 ➜ 返回索引/插入位

关键约束：

  • 数组已排序，这是使用二分查找的前提
  • 时间复杂度必须 O(log n)，不能用线性扫描
  • 找不到时返回插入位置，不是 -1
3 解题思路

核心思想：标准二分查找的变体。当 nums[mid] < target 时，left = mid + 1；否则 right = mid - 1。循环结束后，left 就是插入位置。

初始化 left=0, right=n-1 ➜ mid = (left+right)/2 ➜ 比较 nums[mid] 与 target ➜ 缩小范围

为什么 left 是插入位置？每次 nums[mid] < target 时 left 右移，最终 left 停在第一个 ≥ target 的位置。

4 代码演示
def searchInsert(nums, target): left, right = 0, len(nums) - 1 while left <= right: mid = (left + right) // 2 if nums[mid] < target: left = mid + 1 else: right = mid - 1 return left # left 即为插入位置
public int searchInsert(int[] nums, int target) { int left = 0, right = nums.length - 1; while (left <= right) { int mid = left + (right - left) / 2; if (nums[mid] < target) { left = mid + 1; } else { right = mid - 1; } } return left; }
int searchInsert(vector<int>& nums, int target) { int left = 0, right = nums.size() - 1; while (left <= right) { int mid = left + (right - left) / 2; if (nums[mid] < target) { left = mid + 1; } else { right = mid - 1; } } return left; }

6 易错要点
循环条件 left ≤ right

必须用 left <= right，不是 <。当 left == right 时还需要判断该位置，否则可能漏掉。

返回 left 不是 mid

循环结束后 left 指向第一个 ≥ target 的位置，就是插入点。mid 在循环外已无意义。

防溢出写法

计算 mid 时用 left + (right - left) / 2 而非 (left + right) / 2，避免大数相加溢出。

7 扩展思考
  • lower_bound / upper_bound：本题本质是 C++ 的 lower_bound，找第一个 ≥ target 的位置
  • 如果允许重复元素：返回最左边的插入位置，逻辑不变
  • 变体：找最后一个 ≤ target 的位置：即 upper_bound - 1
  • 二分模板统一：掌握「找左边界」和「找右边界」两套模板，覆盖所有二分题
