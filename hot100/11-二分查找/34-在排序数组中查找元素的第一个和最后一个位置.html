#34 查找首末位置

给定一个按照升序排列的整数数组 nums 和目标值 target，找出给定目标值在数组中的开始位置和结束位置。如果数组中不存在目标值，返回 [-1, -1]。必须设计并实现时间复杂度为 O(log n) 的算法。

示例 1
输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]
示例 2
输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]
示例 3
输入：nums = [], target = 0
输出：[-1,-1]
2 题目拆解
排序数组 ➜ 可能有重复元素 ➜ 找左边界 + 右边界 ➜ 返回 [first, last]

关键约束：

  • 数组升序，但可能有重复元素
  • 要求 O(log n)，不能线性扫描
  • 找不到返回 [-1, -1]
3 解题思路

两次二分：第一次找左边界（lower_bound），第二次找右边界（upper_bound - 1）。

找左边界 ➜ nums[mid] ≥ target → right=mid-1 ➜ 找右边界 ➜ nums[mid] ≤ target → left=mid+1

找左边界：当 nums[mid] >= target 时收缩右边界，最终 left 指向第一个等于 target 的位置。

找右边界：当 nums[mid] <= target 时收缩左边界，最终 right 指向最后一个等于 target 的位置。

4 代码演示
def searchRange(nums, target): def findLeft(): l, r = 0, len(nums) - 1 while l <= r: m = (l + r) // 2 if nums[m] >= target: r = m - 1 else: l = m + 1 return l def findRight(): l, r = 0, len(nums) - 1 while l <= r: m = (l + r) // 2 if nums[m] <= target: l = m + 1 else: r = m - 1 return r if not nums: return [-1, -1] left, right = findLeft(), findRight() return [left, right] if left <= right else [-1, -1]
public int[] searchRange(int[] nums, int target) { if (nums.length == 0) return new int[]{-1, -1}; int left = findLeft(nums, target); int right = findRight(nums, target); return left <= right ? new int[]{left, right} : new int[]{-1, -1}; } int findLeft(int[] nums, int t) { int l = 0, r = nums.length - 1; while (l <= r) { int m = l+(r-l)/2; if (nums[m] >= t) r = m-1; else l = m+1; } return l; } int findRight(int[] nums, int t) { int l = 0, r = nums.length - 1; while (l <= r) { int m = l+(r-l)/2; if (nums[m] <= t) l = m+1; else r = m-1; } return r; }
vector<int> searchRange(vector<int>& nums, int target) { if (nums.empty()) return {-1, -1}; auto left = lower_bound(nums.begin(), nums.end(), target); auto right = upper_bound(nums.begin(), nums.end(), target); if (left == right) return {-1, -1}; return {( int)(left - nums.begin()), (int)(right - nums.begin() - 1)}; }

6 易错要点
两次二分方向不同

找左边界：nums[mid] >= target 时右缩；找右边界：nums[mid] <= target 时左缩。方向搞反结果就错。

left > right 说明不存在

如果 target 不在数组中，findLeft 返回的 left 会大于 findRight 返回的 right，此时应返回 [-1,-1]。

C++ 可直接用 STL

lower_bound 返回第一个 ≥ target 的迭代器，upper_bound 返回第一个 > target 的迭代器。两者相等说明不存在。

7 扩展思考
  • 统计出现次数：upper_bound - lower_bound 即为 target 出现次数，O(log n)
  • 只出现一次的元素：如果所有元素都出现两次，只有一个出现一次，用异或更简单
  • 旋转数组中的查找：数组旋转后不再全局有序，需要先判断 mid 在哪一半
