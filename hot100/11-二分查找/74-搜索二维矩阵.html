#74 搜索二维矩阵

编写一个高效的算法来判断 m x n 矩阵中是否存在一个目标值。该矩阵具有如下特性：每行中的整数从左到右按升序排列；每行的第一个整数大于前一行的最后一个整数。

示例 1
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true
示例 2
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
输出：false
2 题目拆解
行有序矩阵 ➜ 展平为一维 ➜ 二分查找 ➜ 坐标映射回矩阵

关键约束：

  • 每行从左到右递增，行首 > 上一行行尾 → 可视为一维有序数组
  • 要求O(log(m×n))，不能逐行扫描
  • 一维索引 idx → 二维坐标 (idx/n, idx%n)
3 解题思路

核心思想：将 m×n 矩阵视为长度为 m×n 的一维有序数组，直接二分查找。通过 idx / n 和 idx % n 在一维索引和二维坐标之间转换。

left=0, right=m×n-1 ➜ mid → (row, col) ➜ 比较 matrix[row][col] ➜ 缩小范围
4 代码演示
def searchMatrix(matrix, target): m, n = len(matrix), len(matrix[0]) left, right = 0, m * n - 1 while left <= right: mid = (left + right) // 2 row, col = mid // n, mid % n if matrix[row][col] == target: return True elif matrix[row][col] < target: left = mid + 1 else: right = mid - 1 return False
public boolean searchMatrix(int[][] matrix, int target) { int m = matrix.length, n = matrix[0].length; int left = 0, right = m * n - 1; while (left <= right) { int mid = left + (right - left) / 2; int row = mid / n, col = mid % n; if (matrix[row][col] == target) return true; else if (matrix[row][col] < target) left = mid + 1; else right = mid - 1; } return false; }
bool searchMatrix(vector<vector<int>>& matrix, int target) { int m = matrix.size(), n = matrix[0].size(); int left = 0, right = m * n - 1; while (left <= right) { int mid = left + (right - left) / 2; if (matrix[mid/n][mid%n] == target) return true; else if (matrix[mid/n][mid%n] < target) left = mid + 1; else right = mid - 1; } return false; }

6 易错要点
坐标映射

一维 idx → 二维 (idx/n, idx%n)，注意除的是列数 n，不是行数 m。搞反会越界。

矩阵特性判断

本题能用一维二分是因为「行首 > 上一行行尾」。如果只保证行内有序、行间不一定，则不能这样展平。

边界处理

空矩阵 matrix=[] 或 matrix=[[]] 需要特判，否则 matrix[0] 越界。

7 扩展思考
  • 行间不一定有序：用「从右上角开始搜索」的方法，O(m+n)
