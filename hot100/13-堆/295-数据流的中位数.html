#295 数据流的中位数

中位数是有序整数列表中的中间值。如果列表的大小是偶数，则没有中间值，中位数是两个中间值的平均值。设计一个支持以下两种操作的数据结构：addNum(num) — 从数据流中添加一个整数到数据结构中。findMedian() — 返回目前所有元素的中位数。

示例
输入：["MedianFinder","addNum","findMedian","addNum","findMedian","addNum","findMedian"] [[],[1],[],[2],[],[3],[]]
输出：[null,null,1.5,null,2.0,null,2.0]
addNum(1) → findMedian() = 1.5 → addNum(2) → findMedian() = 1.5 → addNum(3) → findMedian() = 2.0
2 题目拆解
数据流 ➜ 动态添加 ➜ 实时中位数 ➜ O(log n) 维护

关键约束：

  • 数据动态添加，不是一次性给出
  • 需要实时查询中位数，每次查询都要快
  • 中位数 = 排序后中间的数（偶数取两个平均值）
  • 最多调用 5 × 10⁴ 次，暴力排序 O(n log n) 太慢
3 解题思路

暴力法 O(n log n)：每次 addNum 后排序，findMedian 取中间 → 每次 findMedian 都要排序，太慢

双堆法 O(log n)：用一个大顶堆存较小的一半，一个小顶堆存较大的一半：

大顶堆（左半） ➜ 中位数 ➜ 小顶堆（右半）

规则：大顶堆元素数 ≤ 小顶堆元素数 ≤ 大顶堆元素数 + 1。这样中位数要么是大顶堆堆顶，要么是两个堆顶的平均。

4 代码演示
import heapq class MedianFinder: def __init__(self): self.maxHeap = [] # 大顶堆（存负数模拟） self.minHeap = [] # 小顶堆 def addNum(self, num): heapq.heappush(self.maxHeap, -num) # 先放大顶堆 heapq.heappush(self.minHeap, -heapq.heappop(self.maxHeap)) # 弹最大→小顶堆 if len(self.minHeap) > len(self.maxHeap): heapq.heappush(self.maxHeap, -heapq.heappop(self.minHeap)) def findMedian(self): if len(self.maxHeap) > len(self.minHeap): return -self.maxHeap[0] return (-self.maxHeap[0] + self.minHeap[0]) / 2
class MedianFinder { PriorityQueue<Integer> maxHeap; // 较小的一半 PriorityQueue<Integer> minHeap; // 较大的一半 public MedianFinder() { maxHeap = new PriorityQueue<>(Collections.reverseOrder()); minHeap = new PriorityQueue<>(); } public void addNum(int num) { maxHeap.offer(num); minHeap.offer(maxHeap.poll()); if (minHeap.size() > maxHeap.size()) { maxHeap.offer(minHeap.poll()); } } public double findMedian() { if (maxHeap.size() > minHeap.size()) return maxHeap.peek(); return (maxHeap.peek() + minHeap.peek()) / 2.0; } }
class MedianFinder { priority_queue<int> maxHeap; // 较小的一半 priority_queue<int,vector<int>,greater<int>> minHeap; // 较大的一半 public: void addNum(int num) { maxHeap.push(num); minHeap.push(maxHeap.top()); maxHeap.pop(); if (minHeap.size() > maxHeap.size()) { maxHeap.push(minHeap.top()); minHeap.pop(); } } double findMedian() { if (maxHeap.size() > minHeap.size()) return maxHeap.top(); return (maxHeap.top() + minHeap.top()) / 2.0; } };

6 易错要点
堆大小平衡

必须保证大顶堆元素数 ≥ 小顶堆元素数，且差值 ≤ 1。否则中位数位置会错。

Python 大顶堆

Python 的 heapq 只支持小顶堆，需要存负数来模拟大顶堆。取值时别忘了再取负。

整数除法陷阱

两个堆顶求平均时，Java/C++ 要用 2.0 而不是 2，否则整数除法会截断小数。

7 扩展思考
  • 如果只需要中位数：可以用快速选择算法 O(n)，但不适合动态数据流
  • 如果数据范围有限：可以用计数排序 + 前缀和，O(1) 查询
  • 滑动窗口中位数：LeetCode 480，在双堆基础上加延迟删除
  • 多路归并：双堆思想可推广到多路归并排序中的选择问题
