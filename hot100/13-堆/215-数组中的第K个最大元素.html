#215 数组中的第K个最大元素

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。

示例
输入：nums = [3,2,1,5,6,4], k = 2
输出：5
排序后 [1,2,3,4,5,6]，第2大的是 5
2 题目拆解
无序数组 ➜ 找第K大 ➜ 不需要完全排序 ➜ 堆/快速选择

关键约束：

  • 要求 O(n) 时间复杂度，排序 O(nlogn) 不满足
  • 第 K 大 = 排序后倒数第 K 个
  • 可能有重复元素，K 是按位置不是按不同值
3 解题思路

方法一：小顶堆 O(nlogk)

维护大小为 k 的小顶堆，堆顶就是第 k 大元素。遍历时，比堆顶大就替换。

方法二：快速选择 O(n) 平均

随机选pivot ➜ partition分区 ➜ pivot位置=target? ➜ 递归一侧

第 K 大 = 第 (n-K) 小，用快速选择找第 (n-K) 小的元素。

4 代码演示
import heapq def findKthLargest(nums, k): # 方法一：小顶堆 heap = [] for num in nums: heapq.heappush(heap, num) if len(heap) > k: heapq.heappop(heap) # 弹出最小的，保留k个最大的 return heap[0] # 堆顶即第k大
public int findKthLargest(int[] nums, int k) { // 方法二：快速选择 int target = nums.length - k; // 第k大 = 第(n-k)小 return quickSelect(nums, 0, nums.length - 1, target); } int quickSelect(int[] nums, int lo, int hi, int target) { int pivot = nums[hi], p = lo; for (int i = lo; i < hi; i++) if (nums[i] <= pivot) swap(nums, p++, i); swap(nums, p, hi); if (p == target) return nums[p]; return p < target ? quickSelect(nums, p+1, hi, target) : quickSelect(nums, lo, p-1, target); }
int findKthLargest(vector<int>& nums, int k) { // 小顶堆 priority_queue<int,vector<int>,greater<int>> pq; for (int num : nums) { pq.push(num); if (pq.size() > k) pq.pop(); } return pq.top(); }

6 易错要点
小顶堆不是大顶堆

求第 K 大用小顶堆（堆顶最小），维护 k 个最大元素，堆顶就是答案。大顶堆需要全部入堆再弹出 k-1 次。

快速选择最坏 O(n²)

如果每次 pivot 选到极端值，快速选择退化为 O(n²)。随机化 pivot 可以期望 O(n)，但最坏仍可能超时。

第K大 vs 第K小

第 K 大 = 排序后倒数第 K 个 = 第 (n-K+1) 小。快速选择时注意转换：target = n - k。

7 扩展思考
  • BFPRT算法：中位数的中位数选 pivot，保证最坏 O(n)，但常数大，实际不一定更快
  • 流式数据：数据量极大无法全部加载时，堆方法更实用（只需维护 k 大小的堆）
  • 前K个高频元素：LeetCode #347，先用哈希统计频率，再用堆取 TopK
  • 中位数：LeetCode #295，用大小堆各维护一半，实现动态中位数
