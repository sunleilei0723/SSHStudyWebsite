#560 和为K的子数组

给定一个整数数组 nums 和一个整数 k，请你统计并返回数组中和为 k 的子数组的个数。子数组是数组中元素的连续非空序列。

2 题目拆解
输入数组 ➜ 目标值 k ➜ 计算前缀和 ➜ 查哈希表计数 ➜ 返回总数

关键约束：

  • 子数组必须是连续的，不是子序列
  • 元素可以为负数，不能滑动窗口
  • 需要统计所有满足条件的子数组个数
3 解题思路

暴力法 O(n²)：枚举所有子数组，逐个求和判断 → 太慢，淘汰

前缀和 + 哈希表 O(n)：定义前缀和 pre[i] = nums[0] + ... + nums[i-1]，则子数组 nums[j..i] 的和 = pre[i+1] - pre[j]。若 pre[i+1] - pre[j] = k，即 pre[j] = pre[i+1] - k。用哈希表记录每个前缀和出现的次数：

遍历计算前缀和 ➜ 查 pre - k 是否在哈希表 ➜ 累加计数 ➜ 更新哈希表
4 代码演示
def subarraySum(nums, k): count = 0 pre_sum = 0 prefix_count = {0: 1} # 前缀和0出现1次 for num in nums: pre_sum += num # 累加前缀和 if (pre_sum - k) in prefix_count: count += prefix_count[pre_sum - k] # 累加计数 prefix_count[pre_sum] = prefix_count.get(pre_sum, 0) + 1 return count
public int subarraySum(int[] nums, int k) { int count = 0, preSum = 0; Map<Integer, Integer> map = new HashMap<>(); map.put(0, 1); for (int num : nums) { preSum += num; if (map.containsKey(preSum - k)) count += map.get(preSum - k); map.put(preSum, map.getOrDefault(preSum, 0) + 1); } return count; }
int subarraySum(vector<int>& nums, int k) { int count = 0, preSum = 0; unordered_map<int, int> mp; mp[0] = 1; for (int num : nums) { preSum += num; if (mp.count(preSum - k)) count += mp[preSum - k]; mp[preSum]++; } return count; }

6 易错要点
初始化 {0:1}

哈希表必须初始化 {0:1}，表示前缀和为0出现1次。否则当 pre_sum == k 时会漏算从开头到当前的子数组。

不能滑动窗口

因为有负数，前缀和不单调递增，窗口收缩不能保证正确性。必须用前缀和+哈希表。

先查再存

必须先查 pre_sum - k 再更新哈希表，否则会把当前前缀和也算进去，导致逻辑错误。

7 扩展思考
  • 如果全为正数：可以用滑动窗口，O(n) 时间 O(1) 空间
  • 和为K的最长子数组：哈希表只记录前缀和第一次出现的位置
  • 和为K的最短子数组：哈希表记录前缀和最后一次出现的位置
  • 二维矩阵版本：LeetCode 1074，前缀和+哈希表的二维扩展
