#03 无重复字符的最长子串

给定一个字符串 s，找出不含有重复字符的最长子串的长度。子串是连续的字符序列。

示例 1
输入：s = "abcabcbb"
输出：3
因为无重复字符的最长子串是 "abc"
示例 2
输入：s = "bbbbb"
输出：1
示例 3
输入：s = "pwwkew"
输出：3
最长子串是 "wke"
2 题目拆解
输入字符串 ➜ 找最长子串 ➜ 无重复字符 ➜ 返回长度

关键约束：

  • 子串必须是连续的，不是子序列
  • "无重复"意味着窗口内每个字符只出现一次
  • 只需要返回长度，不需要返回子串本身
  • s 由英文字母、数字、符号和空格组成
3 解题思路

暴力法 O(n²)：枚举所有子串，检查是否有重复 → 太慢，淘汰

滑动窗口 O(n)：维护一个窗口 [left, right]，用哈希集合记录窗口内的字符。right 右移遇到重复时，left 右移直到窗口无重复：

right 右移扩展窗口 ➜ s[right] 在窗口中？ ➜ 重复→left 右移收缩 ➜ 更新最大长度
4 代码演示
def lengthOfLongestSubstring(s: str) -> int: seen = {} # 字符→最新下标 left = 0 max_len = 0 for right, ch in enumerate(s): if ch in seen and seen[ch] >= left: left = seen[ch] + 1 # 跳到重复字符之后 seen[ch] = right max_len = max(max_len, right - left + 1) return max_len
public int lengthOfLongestSubstring(String s) { Map<Character, Integer> seen = new HashMap<>(); int left = 0, maxLen = 0; for (int right = 0; right < s.length(); right++) { char ch = s.charAt(right); if (seen.containsKey(ch) && seen.get(ch) >= left) { left = seen.get(ch) + 1; } seen.put(ch, right); maxLen = Math.max(maxLen, right - left + 1); } return maxLen; }
int lengthOfLongestSubstring(string s) { unordered_map<char, int> seen; int left = 0, maxLen = 0; for (int right = 0; right < s.size(); right++) { if (seen.count(s[right]) && seen[s[right]] >= left) { left = seen[s[right]] + 1; } seen[s[right]] = right; maxLen = max(maxLen, right - left + 1); } return maxLen; }

6 易错要点
left 跳跃而非逐步

遇到重复字符时，left 应直接跳到 seen[ch] + 1，而不是一步一步右移。逐步右移虽然结果正确但效率退化为 O(n²)。

seen[ch] >= left 判断

必须检查 seen[ch] >= left，因为哈希表中可能存着窗口外的旧下标。不加此判断会误判为重复，导致 left 错误回退。

先移 left 再更新 seen

必须先根据重复判断移动 left，再更新 seen[ch] = right。顺序反了会导致 left 跳过正确位置。

7 扩展思考
  • 如果字符集有限（如仅小写字母）：可用长度 26 的数组代替 HashMap，效率更高
  • 如果要求返回子串本身：额外记录最大长度对应的起止位置，最后截取
  • 至多重复 K 次的最长子串：将"无重复"推广为"至多 K 次"，用计数器替代集合
  • 两个字符串的公共子串：可结合滑动窗口与哈希，逐位置匹配
