#155 最小栈

设计一个支持 push，pop，top 操作，并能在常数时间内检索到最小元素的栈。实现 MinStack 类：push(val) 将元素入栈，pop() 移除栈顶元素，top() 获取栈顶元素，getMin() 获取栈中最小元素。

示例
输入：["MinStack","push","push","push","getMin","pop","top","getMin"]
参数：[[],[-2],[0],[-3],[],[],[],[]]
输出：[null,null,null,null,-3,null,0,-2]
push(-2) → push(0) → push(-3) → getMin()=-3 → pop() → top()=0 → getMin()=-2
2 题目拆解
常规栈操作 ➜ O(1)获取最小值 ➜ 辅助栈同步 ➜ pop时同步弹出

关键约束：

  • getMin() 必须 O(1)，不能遍历
  • 所有操作都是栈顶操作，符合 LIFO
  • 最小值可能随 pop 而变化
3 解题思路

双栈法：一个主栈存数据，一个辅助栈存每个位置对应的最小值。两个栈同步 push/pop。

push(val) ➜ 主栈入val ➜ 辅助栈入min(栈顶, val) ➜ getMin() → 辅助栈顶

优化：辅助栈只在最小值变化时入栈（存 ≤ 当前最小值 的值），pop 时检查是否需要同步弹出。

4 代码演示
class MinStack: def __init__(self): self.stack = [] self.min_stack = [] # 辅助栈：同步记录最小值 def push(self, val): self.stack.append(val) self.min_stack.append(min(val, self.min_stack[-1] if self.min_stack else val)) def pop(self): self.stack.pop() self.min_stack.pop() def top(self): return self.stack[-1] def getMin(self): return self.min_stack[-1]
class MinStack { Deque<Integer> stack = new ArrayDeque<>(); Deque<Integer> minStack = new ArrayDeque<>(); public void push(int val) { stack.push(val); minStack.push(minStack.isEmpty() ? val : Math.min(val, minStack.peek())); } public void pop() { stack.pop(); minStack.pop(); } public int top() { return stack.peek(); } public int getMin() { return minStack.peek(); } }
class MinStack { vector<int> stk, minStk; public: void push(int val) { stk.push_back(val); minStk.push_back(minStk.empty() ? val : min(val, minStk.back())); } void pop() { stk.pop_back(); minStk.pop_back(); } int top() { return stk.back(); } int getMin() { return minStk.back(); } };

6 易错要点
同步弹出

pop() 时主栈和辅助栈必须同时弹出。只弹主栈会导致辅助栈记录的最小值与实际不一致。

辅助栈初始化

第一个元素入栈时，辅助栈为空，min_stack[-1] 会越界。需要特判：空时直接入 val。

相等也要入栈

如果 val == minStack.top()，辅助栈也要入栈！否则 pop 时会把最小值错误弹出。如连续两个 -3。

7 扩展思考
  • 空间优化：辅助栈只在最小值变化时入栈（≤ 当前最小值），pop 时检查是否等于辅助栈顶
  • 最大栈：同理，再加一个辅助栈记录最大值即可
  • 差值法：主栈存 val - min 的差值，用一个变量记录当前最小值，空间 O(1) 但有溢出风险
  • 多线程安全：实际工程中需要加锁保证线程安全
