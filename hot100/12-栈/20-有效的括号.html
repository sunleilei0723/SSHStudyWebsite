#20 有效的括号

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。有效字符串需满足：左括号必须用相同类型的右括号闭合，左括号必须以正确的顺序闭合，每个右括号都有一个对应的相同类型的左括号。

示例 1
输入：s = "()"
输出：true
示例 2
输入：s = "()[]{}"
输出：true
示例 3
输入：s = "(]"
输出：false
2 题目拆解
遍历字符串 ➜ 左括号入栈 ➜ 右括号匹配栈顶 ➜ 栈空则有效

关键约束：

  • 括号必须成对出现且嵌套正确
  • 遇到右括号时，栈顶必须是对应的左括号
  • 最终栈必须为空，否则有多余左括号
3 解题思路

栈的核心思想：最近遇到的左括号需要最先被匹配 → 后进先出(LIFO)，天然适合栈！

遇到左括号 ➜ 入栈等待匹配 ➜ 遇到右括号 ➜ 检查栈顶是否匹配

技巧：遇到左括号时，直接将对应的右括号入栈，这样匹配时只需比较栈顶和当前字符是否相等，省去映射表查找。

4 代码演示
def isValid(s: str) -> bool: stack = [] mapping = {'(': ')', '[': ']', '{': '}'} for ch in s: if ch in mapping: # 左括号 → 入栈对应的右括号 stack.append(mapping[ch]) elif not stack or stack.pop() != ch: # 右括号 → 检查匹配 return False return not stack # 栈空则有效
public boolean isValid(String s) { Deque<Character> stack = new ArrayDeque<>(); for (char ch : s.toCharArray()) { if (ch == '(') stack.push(')'); else if (ch == '[') stack.push(']'); else if (ch == '{') stack.push('}'); else if (stack.isEmpty() || stack.pop() != ch) return false; } return stack.isEmpty(); }
bool isValid(string s) { vector<char> stk; for (char ch : s) { if (ch == '(') stk.push_back(')'); else if (ch == '[') stk.push_back(']'); else if (ch == '{') stk.push_back('}'); else if (stk.empty() || stk.back() != ch) return false; else stk.pop_back(); } return stk.empty(); }

6 易错要点
空栈判断

遇到右括号时，必须先检查栈是否为空。如果栈空说明没有对应的左括号，直接返回 false。

最终栈检查

遍历结束后，栈必须为空。如果栈非空，说明有未匹配的左括号，如 "([" 应返回 false。

入栈右括号技巧

遇到左括号时直接入栈对应的右括号，匹配时只需 pop() != ch，无需额外映射查找，代码更简洁。
