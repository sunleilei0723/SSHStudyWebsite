#84 柱状图中最大的矩形

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。求在该柱状图中，能够勾勒出来的最大矩形面积。

示例
输入：heights = [2,1,5,6,2,3]
输出：10
最大矩形面积为 10，由高度 5 和 6 的两个柱子组成（宽度2×高度5）
2 题目拆解
每根柱子为高 ➜ 找左右边界 ➜ 计算面积=高×宽 ➜ 取最大值

关键约束：

  • 矩形高度由最矮的柱子决定
  • 对每根柱子，需要找左右第一个比它矮的位置，确定宽度
  • 暴力 O(n²) 太慢，单调栈 O(n)
3 解题思路

单调栈：维护从栈底到栈顶递增的栈。当当前柱子比栈顶矮时，栈顶柱子的右边界确定了，左边界是栈顶下一个元素。

遍历 heights ➜ h[i] < h[栈顶]? ➜ 弹出，计算面积 ➜ 宽=右边界-左边界-1

技巧：首尾加 0（哨兵），确保所有柱子都能被弹出计算。

4 代码演示
def largestRectangleArea(heights): heights = [0] + heights + [0] # 前后加哨兵 stack = [] max_area = 0 for i in range(len(heights)): while stack and heights[i] < heights[stack[-1]]: h = heights[stack.pop()] # 弹出柱子的高度 w = i - stack[-1] - 1 # 宽度 = 右边界 - 左边界 - 1 max_area = max(max_area, h * w) stack.append(i) return max_area
public int largestRectangleArea(int[] heights) { int[] h = new int[heights.length + 2]; System.arraycopy(heights, 0, h, 1, heights.length); // 前后哨兵0 Deque<Integer> stack = new ArrayDeque<>(); int maxArea = 0; for (int i = 0; i < h.length; i++) { while (!stack.isEmpty() && h[i] < h[stack.peek()]) { int height = h[stack.pop()]; int width = i - stack.peek() - 1; maxArea = Math.max(maxArea, height * width); } stack.push(i); } return maxArea; }
int largestRectangleArea(vector<int>& heights) { heights.insert(heights.begin(), 0); heights.push_back(0); // 前后哨兵 vector<int> stk; int maxArea = 0; for (int i = 0; i < heights.size(); i++) { while (!stk.empty() && heights[i] < heights[stk.back()]) { int h = heights[stk.back()]; stk.pop_back(); int w = i - stk.back() - 1; maxArea = max(maxArea, h * w); } stk.push_back(i); } return maxArea; }

6 易错要点
哨兵必须加

首尾加 0 确保所有柱子都能被弹出计算。不加哨兵，最后的递增序列不会被处理，导致漏算。

宽度计算

w = i - stack[-1] - 1，不是 i - popped。左边界是弹出后新的栈顶，不是被弹出元素本身。

栈存下标

和每日温度一样，栈存下标不存值。弹出后需要用 stack[-1] 作为左边界，只有存下标才能做到。
