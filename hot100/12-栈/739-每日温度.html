#739 每日温度

给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 0 来代替。

示例
输入：temperatures = [73,74,75,71,69,72,76,73]
输出：[1,1,4,2,1,1,0,0]
第0天73°，第1天74°更高，1-0=1天；第2天75°，第6天76°更高，6-2=4天
2 题目拆解
遍历温度数组 ➜ 找下一个更高温度 ➜ 计算天数差 ➜ 无更高→0

关键约束：

  • 要找的是下一个更高温度，不是全局最高
  • 返回的是天数差（下标差），不是温度值
  • 暴力法 O(n²) 太慢，需要 O(n)
3 解题思路

单调栈：维护一个从栈底到栈顶递减的栈（存下标）。当前温度比栈顶高时，弹出栈顶并计算天数差。

当前温度 T[i] ➜ T[i] > T[栈顶]? ➜ 弹出，ans[栈顶]=i-栈顶 ➜ 当前下标入栈

核心：栈中存的是还没找到答案的下标，栈保持递减意味着这些天温度依次降低，等待被"解救"。

4 代码演示
def dailyTemperatures(temperatures): n = len(temperatures) ans = [0] * n stack = [] # 单调递减栈，存下标 for i in range(n): while stack and temperatures[i] > temperatures[stack[-1]]: prev = stack.pop() ans[prev] = i - prev # 天数差 stack.append(i) return ans # 栈中剩余的默认为0
public int[] dailyTemperatures(int[] temperatures) { int n = temperatures.length; int[] ans = new int[n]; Deque<Integer> stack = new ArrayDeque<>(); for (int i = 0; i < n; i++) { while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) { int prev = stack.pop(); ans[prev] = i - prev; } stack.push(i); } return ans; }
vector<int> dailyTemperatures(vector<int>& T) { int n = T.size(); vector<int> ans(n, 0); vector<int> stk; for (int i = 0; i < n; i++) { while (!stk.empty() && T[i] > T[stk.back()]) { int prev = stk.back(); stk.pop_back(); ans[prev] = i - prev; } stk.push_back(i); } return ans; }

6 易错要点
栈存下标

栈中存的是下标不是温度值。因为需要计算天数差 i - prev，只有存下标才能算。

while 不是 if

当前温度可能比栈中多个元素都高，需要用 while 循环弹出所有比当前低的栈顶，不能只弹一次。

默认值为0

初始化 ans 全为 0，栈中剩余元素不需要处理，它们后面没有更高温度，答案就是 0。
