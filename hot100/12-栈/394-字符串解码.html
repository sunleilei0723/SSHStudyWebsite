#394 字符串解码

给定一个经过编码的字符串，返回它解码后的字符串。编码规则为 k[encoded_string]，表示其中 encoded_string 正好重复 k 次。注意 k 保证为正整数。你可以认为输入字符串总是有效的，且不存在额外空格。

示例 1
输入：s = "3[a]2[bc]"
输出："aaabcbc"
示例 2
输入：s = "3[a2[c]]"
输出："accaccacc"
2 题目拆解
遍历字符串 ➜ 数字→记录倍数 ➜ [ → 入栈保存 ➜ ] → 解码出栈

关键约束：

  • 可能有嵌套，如 3[a2[c]]，需要逐层解码
  • k 可能是多位数，如 10[a]
  • 遇到 [ 时需要保存当前状态，进入新层级
3 解题思路

双栈法：一个栈存倍数，一个栈存之前的字符串。遇到 [ 时保存现场，遇到 ] 时解码出栈。

数字 ➜ 解析完整k，入num栈 ➜ [ ➜ 当前串入str栈，重置 ➜ ] ➜ 重复k次+拼接str栈顶
4 代码演示
def decodeString(s: str) -> str: num_stack = [] # 数字栈 str_stack = [] # 字符串栈 cur_num = 0 cur_str = '' for ch in s: if ch.isdigit(): cur_num = cur_num * 10 + int(ch) # 多位数处理 elif ch == '[': num_stack.append(cur_num); str_stack.append(cur_str) cur_num = 0; cur_str = '' # 保存现场，重置 elif ch == ']': cur_str = str_stack.pop() + num_stack.pop() * cur_str # 解码 else: cur_str += ch # 普通字母追加 return cur_str
public String decodeString(String s) { Deque<Integer> numStack = new ArrayDeque<>(); Deque<String> strStack = new ArrayDeque<>(); int curNum = 0; String curStr = ""; for (char ch : s.toCharArray()) { if (Character.isDigit(ch)) curNum = curNum * 10 + (ch - '0'); else if (ch == '[') { numStack.push(curNum); strStack.push(curStr); curNum = 0; curStr = ""; } else if (ch == ']') { int k = numStack.pop(); String prev = strStack.pop(); curStr = prev + curStr.repeat(k); } else curStr += ch; } return curStr; }
string decodeString(string s) { vector<int> numStk; vector<string> strStk; int curNum = 0; string curStr; for (char ch : s) { if (isdigit(ch)) curNum = curNum * 10 + (ch - '0'); else if (ch == '[') { numStk.push_back(curNum); strStk.push_back(curStr); curNum = 0; curStr.clear(); } else if (ch == ']') { int k = numStk.back(); numStk.pop_back(); string prev = strStk.back(); strStk.pop_back(); string rep; for(int i=0;i<k;i++) rep+=curStr; curStr = prev + rep; } else curStr += ch; } return curStr; }

6 易错要点
多位数字

cur_num = cur_num * 10 + int(ch)，不能只取单个数字。如 10[a] 中 10 是两位数。

入栈时机

遇到 [ 时才入栈，不是遇到数字时。数字只是暂存在 cur_num 中，等 [ 确认倍数。

解码拼接顺序

] 时：cur_str = str_stack.pop() + num_stack.pop() * cur_str。注意是之前的串 + 重复后的当前串，顺序不能反。

7 扩展思考
  • 递归解法：遇到 [ 递归处理子串，遇到 ] 返回，天然处理嵌套
  • 更多编码规则：如果支持 k{str} 或 k<str>，只需扩展括号匹配逻辑
  • 编码问题：给定字符串，如何编码为最短形式？贪心找重复模式
  • 嵌套深度限制：如果嵌套深度有限制，递归解法需要加深度检查
