#78 子集

给定一个不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。解集不能包含重复的子集。

示例 1
输入：nums = [1,2,3]
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
示例 2
输入：nums = [0]
输出：[[],[0]]
2 题目拆解
不含重复的数组 ➜ 枚举所有子集 ➜ 返回幂集

关键约束：

  • 数组不含重复元素，子集天然不重复
  • 子集无序：[1,2] 和 [2,1] 是同一子集
  • 空集 [] 也是合法子集
  • n 个元素有 2ⁿ 个子集
3 解题思路

方法一：回溯法 — 每个元素「选 / 不选」，用 start 索引避免重复：

从 start 开始遍历 ➜ 选 nums[i] 加入路径 ➜ 递归 i+1 ➜ 撤销，继续

方法二：位运算 — 用 0~2ⁿ-1 的二进制表示选取状态，1=选，0=不选。

4 代码演示
def subsets(nums): res = [] def backtrack(start, path): res.append(path[:]) # 每个节点都收集 for i in range(start, len(nums)): path.append(nums[i]) # 选择 backtrack(i + 1, path) # 递归 i+1 path.pop() # 撤销 backtrack(0, []) return res
public List<List<Integer>> subsets(int[] nums) { List<List<Integer>> res = new ArrayList<>(); backtrack(nums, 0, new ArrayList<>(), res); return res; } void backtrack(int[] nums, int start, List<Integer> path, List<List<Integer>> res) { res.add(new ArrayList<>(path)); // 每个节点收集 for (int i = start; i < nums.length; i++) { path.add(nums[i]); backtrack(nums, i + 1, path, res); path.remove(path.size() - 1); } }
vector<vector<int>> subsets(vector<int>& nums) { vector<vector<int>> res; vector<int> path; backtrack(nums, 0, path, res); return res; } void backtrack(vector<int>& nums, int start, vector<int>& path, vector<vector<int>>& res) { res.push_back(path); // 每个节点收集 for (int i = start; i < nums.size(); i++) { path.push_back(nums[i]); backtrack(nums, i + 1, path, res); path.pop_back(); } }

6 易错要点
每个节点都收集

与全排列不同，子集问题在每个节点（不只是叶节点）都要收集结果，因为每个路径都是合法子集。

start 索引防重复

用 backtrack(i+1, path) 而非 backtrack(start+1, path)，确保只向后选，避免 [2,1] 和 [1,2] 重复。

位运算方法

用 mask >> i & 1 判断第 i 位是否选中。注意循环范围是 0 到 2^n - 1，别漏掉 0（空集）。

7 扩展思考
  • 含重复元素：#90 子集 II，需要排序 + 剪枝 if i>start and nums[i]==nums[i-1]: continue
  • 组合问题：子集是组合的特例，组合 = 固定大小的子集
  • 迭代法：从空集开始，每遇到一个数就把已有子集都加上该数
  • 时间复杂度：O(n·2ⁿ)，2ⁿ 个子集，每个最多 O(n) 拷贝
