#51 N皇后

按照国际象棋规则，皇后可以攻击同行、同列、同对角线上的棋子。给定整数 n，返回所有不同的 n 皇后问题的解决方案。每种方案包含一种不同的 n 皇后棋子放置方案，其中 'Q' 和 '.' 分别代表皇后和空位。

示例 1
输入：n = 4
输出：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
两种解法，4个皇后互不攻击
示例 2
输入：n = 1
输出：[["Q"]]
2 题目拆解
n×n 棋盘 ➜ 放 n 个皇后 ➜ 互不攻击 ➜ 返回所有方案

冲突检测（皇后攻击范围）：

  • 同列：cols 集合记录已占列
  • 主对角线：行-列 = 常数 → diag1 集合记录 row-col
  • 副对角线：行+列 = 常数 → diag2 集合记录 row+col
3 解题思路

逐行回溯：每行恰好放一个皇后，枚举该行的每一列，检查是否冲突，不冲突则放置并递归下一行。

处理第 row 行 ➜ 枚举每列 col ➜ 检查冲突 ➜ 放置皇后，递归 ➜ 撤销，试下一列

终止条件：row == n，收集一个完整方案。

4 代码演示
def solveNQueens(n): res = [] cols, diag1, diag2 = set(), set(), set() def backtrack(row, queens): if row == n: # 放满 n 行 board = [] for q in queens: board.append('.'*q + 'Q' + '.'*(n-q-1)) res.append(board) return for col in range(n): if col in cols or row-col in diag1 or row+col in diag2: continue # 冲突，跳过 queens.append(col) cols.add(col); diag1.add(row-col); diag2.add(row+col) backtrack(row+1, queens) queens.pop() cols.remove(col); diag1.remove(row-col); diag2.remove(row+col) backtrack(0, []) return res
public List<List<String>> solveNQueens(int n) { List<List<String>> res = new ArrayList<>(); backtrack(n, 0, new ArrayList<>(), new HashSet<>(), new HashSet<>(), new HashSet<>(), res); return res; } void backtrack(int n, int row, List<Integer> queens, Set<Integer> cols, Set<Integer> d1, Set<Integer> d2, List<List<String>> res) { if (row == n) { List<String> board = new ArrayList<>(); for (int q : queens) { char[] row2 = new char[n]; Arrays.fill(row2, '.'); row2[q] = 'Q'; board.add(new String(row2)); } res.add(board); return; } for (int col = 0; col < n; col++) { if (cols.contains(col)||d1.contains(row-col)||d2.contains(row+col)) continue; queens.add(col); cols.add(col); d1.add(row-col); d2.add(row+col); backtrack(n, row+1, queens, cols, d1, d2, res); queens.remove(queens.size()-1); cols.remove(col); d1.remove(row-col); d2.remove(row+col); } }
vector<vector<string>> solveNQueens(int n) { vector<vector<string>> res; vector<int> queens; unordered_set<int> cols, d1, d2; backtrack(n, 0, queens, cols, d1, d2, res); return res; } void backtrack(int n, int row, vector<int>& queens, unordered_set<int>& cols, unordered_set<int>& d1, unordered_set<int>& d2, vector<vector<string>>& res) { if (row == n) { vector<string> board; for (int q : queens) { string r(n, '.'); r[q] = 'Q'; board.push_back(r); } res.push_back(board); return; } for (int col = 0; col < n; col++) { if (cols.count(col)||d1.count(row-col)||d2.count(row+col)) continue; queens.push_back(col); cols.insert(col); d1.insert(row-col); d2.insert(row+col); backtrack(n, row+1, queens, cols, d1, d2, res); queens.pop_back(); cols.erase(col); d1.erase(row-col); d2.erase(row+col); } }

6 易错要点
对角线公式

主对角线 row - col，副对角线 row + col。同一对角线上的格子计算值相同。别搞反了！

三个集合都要撤销

回溯时 cols、diag1、diag2 三个集合都要 remove，漏掉任何一个都会导致后续判断错误。

输出格式

题目要求输出字符串列表 [".Q..","...Q"]，不是坐标列表。别忘了把 queens 数组转成棋盘字符串。

7 扩展思考
  • #52 N皇后 II：只求方案数，不需要构造棋盘，可以用位运算加速
  • 位运算优化：用整数的 bit 表示列和对角线占用，O(1) 判断冲突
  • N皇后可解条件：n=2,3 无解，n≥4 都有解
  • 时间复杂度：O(n!)，每行选择受前面行约束，实际远小于 n!
