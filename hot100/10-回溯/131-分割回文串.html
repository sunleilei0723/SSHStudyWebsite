#131 分割回文串

给定字符串 s，将 s 分割成一些子串，使每个子串都是回文串。返回 s 所有可能的分割方案。

示例 1
输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]
两种分割方式：a|a|b 和 aa|b，每个子串都是回文
示例 2
输入：s = "a"
输出：[["a"]]
2 题目拆解
字符串 s ➜ 枚举分割点 ➜ 判断子串回文 ➜ 收集合法分割

关键约束：

  • 每个子串必须是回文 → 需要高效判断
  • 分割必须覆盖整个字符串，不能遗漏字符
  • 同一位置可以有多种分割方式
3 解题思路

回溯 + DP 预处理：先用 DP 预处理所有子串是否回文，再回溯枚举分割点。

DP 预处理回文表 ➜ 从 start 枚举 end ➜ s[start:end] 是回文? ➜ 是 → 加入路径，递归

DP 预处理：dp[i][j] 表示 s[i..j] 是否回文。dp[i][j] = s[i]==s[j] && (j-i<2 || dp[i+1][j-1])

4 代码演示
def partition(s): n = len(s) # DP 预处理回文表 dp = [[False]*n for _ in range(n)] for j in range(n): for i in range(j+1): dp[i][j] = s[i]==s[j] and (j-i<2 or dp[i+1][j-1]) res = [] def backtrack(start, path): if start == n: # 分割完成 res.append(path[:]) return for end in range(start, n): if dp[start][end]: # 是回文才继续 path.append(s[start:end+1]) backtrack(end+1, path) path.pop() backtrack(0, []) return res
public List<List<String>> partition(String s) { int n = s.length(); boolean[][] dp = new boolean[n][n]; for (int j = 0; j < n; j++) for (int i = 0; i <= j; i++) dp[i][j] = s.charAt(i)==s.charAt(j) && (j-i<2 || dp[i+1][j-1]); List<List<String>> res = new ArrayList<>(); backtrack(s, 0, dp, new ArrayList<>(), res); return res; } void backtrack(String s, int start, boolean[][] dp, List<String> path, List<List<String>> res) { if (start == s.length()) { res.add(new ArrayList<>(path)); return; } for (int end = start; end < s.length(); end++) { if (dp[start][end]) { path.add(s.substring(start, end+1)); backtrack(s, end+1, dp, path, res); path.remove(path.size()-1); } } }
vector<vector<string>> partition(string s) { int n = s.size(); vector<vector<bool>> dp(n, vector<bool>(n)); for (int j = 0; j < n; j++) for (int i = 0; i <= j; i++) dp[i][j] = s[i]==s[j] && (j-i<2 || dp[i+1][j-1]); vector<vector<string>> res; vector<string> path; backtrack(s, 0, dp, path, res); return res; } void backtrack(string& s, int start, vector<vector<bool>>& dp, vector<string>& path, vector<vector<string>>& res) { if (start == s.size()) { res.push_back(path); return; } for (int end = start; end < s.size(); end++) { if (dp[start][end]) { path.push_back(s.substr(start, end-start+1)); backtrack(s, end+1, dp, path, res); path.pop_back(); } } }

6 易错要点
DP 预处理顺序

必须先枚举 j 再枚举 i（外层 j，内层 i≤j），这样计算 dp[i][j] 时 dp[i+1][j-1] 已经算好。如果反过来会出错。

不用 DP 也能做

可以在回溯时直接判断回文，但每次判断 O(n)，总复杂度 O(n·2^n)。DP 预处理 O(n²) 一次，回溯时 O(1) 查表。

substring 边界

Python s[start:end+1]，Java s.substring(start, end+1)，C++ s.substr(start, end-start+1)。注意 +1 别漏。
