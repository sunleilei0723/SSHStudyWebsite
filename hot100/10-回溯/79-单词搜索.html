#79 单词搜索

给定一个 m x n 二维字符网格 board 和字符串 word，如果 word 存在于网格中返回 true。单词必须按字母顺序通过相邻单元格的水平或垂直方向组成，同一单元格字母不能重复使用。

示例 1
输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"
输出：true
路径 A(0,0)→B(0,1)→C(0,2)→C(1,2)→E(1,3)...→D(2,1)
示例 2
输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "SEE"
输出：true
示例 3
输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCB"
输出：false
B 不能重复使用
2 题目拆解
二维字符网格 ➜ 目标单词 ➜ 上下左右搜索 ➜ 找到返回 true

关键约束：

  • 只能水平或垂直移动，不能对角线
  • 同一格子不能重复使用 → 需要 visited 标记
  • 可以从任意起点开始搜索
3 解题思路

DFS + 回溯：遍历每个格子作为起点，若格子字符匹配 word[0]，则向四个方向 DFS 搜索剩余字符。

遍历起点 ➜ 匹配 word[i]? ➜ 标记 visited ➜ 四方向 DFS ➜ 撤销 visited

优化：原地修改 board[i][j] 为特殊字符（如 '#'）代替 visited 数组，节省空间。

4 代码演示
def exist(board, word): m, n = len(board), len(board[0]) def dfs(i, j, k): if k == len(word): # 匹配完成 return True if i<0 or i>=m or j<0 or j>=n or board[i][j]!=word[k]: return False tmp = board[i][j] board[i][j] = '#' # 标记已访问 res = (dfs(i+1,j,k+1) or dfs(i-1,j,k+1) or dfs(i,j+1,k+1) or dfs(i,j-1,k+1)) board[i][j] = tmp # 撤销标记 return res for i in range(m): for j in range(n): if dfs(i, j, 0): return True return False

6 易错要点
撤销标记别忘

DFS 返回后必须恢复 board[i][j] = tmp，否则其他起点的搜索路径会被错误阻断。

边界检查顺序

先检查越界，再检查字符匹配。如果先访问 board[i][j] 再检查越界，会数组越界报错。

原地修改 vs visited

原地修改 board[i][j]='#' 省空间，但会修改原数组。如果要求不修改，用 visited 数组。

7 扩展思考
  • #212 单词搜索 II：多个单词同时搜索，用 Trie + DFS 大幅优化
  • 剪枝优化：先统计 board 和 word 的字符频率，若 word 有 board 没有的字符直接返回 false
  • 搜索方向：从 word 首尾中较少出现的字符端开始搜索，可减少无效搜索
  • 时间复杂度：O(m·n·4^L)，L 为 word 长度，实际远小于此因为剪枝
