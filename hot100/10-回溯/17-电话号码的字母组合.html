#17 电话号码的字母组合

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。答案可以按任意顺序返回。数字到字母的映射与电话按键相同。

示例 1
输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]
示例 2
输入：digits = ""
输出：[]
示例 3
输入：digits = "2"
输出：["a","b","c"]
2 题目拆解
数字字符串 ➜ 映射到字母 ➜ 组合所有可能 ➜ 返回结果

映射表：

  • 2→abc, 3→def, 4→ghi, 5→jkl, 6→mno
  • 7→pqrs, 8→tuv, 9→wxyz

关键约束：

  • 空字符串返回空列表，不是 [""]
  • 每个数字对应 3~4 个字母，组合数 = 各位字母数之积
3 解题思路

回溯法：逐位处理数字字符串，对当前数字对应的所有字母枚举选择，递归处理下一位。

处理第 i 位数字 ➜ 枚举该数字的每个字母 ➜ 加入路径，递归 i+1 ➜ 撤销，选下一个字母

终止条件：i == len(digits)，收集一个完整组合。

4 代码演示
def letterCombinations(digits): if not digits: return [] mapping = { '2':'abc','3':'def','4':'ghi','5':'jkl', '6':'mno','7':'pqrs','8':'tuv','9':'wxyz' } res = [] def backtrack(i, path): if i == len(digits): # 终止 res.append(''.join(path)) return for ch in mapping[digits[i]]: path.append(ch) # 选择 backtrack(i + 1, path) # 递归 path.pop() # 撤销 backtrack(0, []) return res
public List<String> letterCombinations(String digits) { if (digits.isEmpty()) return new ArrayList<>(); String[] map = {"","","abc","def","ghi", "jkl","mno","pqrs","tuv","wxyz"}; List<String> res = new ArrayList<>(); backtrack(digits, map, 0, new StringBuilder(), res); return res; } void backtrack(String digits, String[] map, int idx, StringBuilder path, List<String> res) { if (idx == digits.length()) { res.add(path.toString()); return; } for (char c : map[digits.charAt(idx)-'0'].toCharArray()) { path.append(c); backtrack(digits, map, idx+1, path, res); path.deleteCharAt(path.length()-1); } }
vector<string> letterCombinations(string digits) { if (digits.empty()) return {}; string map[] = {"","","abc","def","ghi", "jkl","mno","pqrs","tuv","wxyz"}; vector<string> res; string path; backtrack(digits, map, 0, path, res); return res; } void backtrack(string& digits, string map[], int idx, string& path, vector<string>& res) { if (idx == digits.size()) { res.push_back(path); return; } for (char c : map[digits[idx]-'0']) { path += c; backtrack(digits, map, idx+1, path, res); path.pop_back(); } }

6 易错要点
空字符串边界

digits = "" 应返回 []，不是 [""]。必须在函数开头做特判。

映射表别漏 7 和 9

7→pqrs 和 9→wxyz 是 4 个字母，其余是 3 个。映射表写错会导致结果不全。

字符串拼接效率

Java 中用 StringBuilder，递归后 deleteCharAt 撤销。Python 用 list + join 更高效。

7 扩展思考
  • 迭代法：用队列实现，每次取出已有组合，追加当前数字的所有字母
  • 如果数字含 0/1：0 和 1 没有对应字母，需要特殊处理或跳过
  • 如果要求去重：用 Set 收集结果即可
  • 推广：任意多组选择做笛卡尔积，都可以用回溯
