#22 括号生成

给定 n 对括号，生成所有有效的括号组合。

示例 1
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
示例 2
输入：n = 1
输出：["()"]
2 题目拆解
n 对括号 ➜ 构造有效组合 ➜ 返回全部

有效括号的约束：

  • 左括号数量不超过 n：open < n 时才能放 (
  • 右括号数量不超过左括号：close < open 时才能放 )
  • 终止：open == close == n
3 解题思路

DFS + 剪枝：在每一步决定放 ( 还是 )，通过 open/close 计数器约束合法性。

open < n → 可放 ( ➜ close < open → 可放 ) ➜ open==close==n → 收集

这不需要「撤销选择」的传统回溯，因为字符串是不可变的，每次递归创建新串。但思路本质相同。

4 代码演示
def generateParenthesis(n): res = [] def dfs(path, open, close): if open == close == n: # 终止 res.append(path) return if open < n: # 可以放 ( dfs(path + '(', open + 1, close) if close < open: # 可以放 ) dfs(path + ')', open, close + 1) dfs('', 0, 0) return res
public List<String> generateParenthesis(int n) { List<String> res = new ArrayList<>(); dfs("", n, 0, 0, res); return res; } void dfs(String path, int n, int open, int close, List<String> res) { if (open == close && open == n) { res.add(path); return; } if (open < n) dfs(path + "(", n, open+1, close, res); if (close < open) dfs(path + ")", n, open, close+1, res); }
vector<string> generateParenthesis(int n) { vector<string> res; dfs("", n, 0, 0, res); return res; } void dfs(string path, int n, int open, int close, vector<string>& res) { if (open == close && open == n) { res.push_back(path); return; } if (open < n) dfs(path + "(", n, open+1, close, res); if (close < open) dfs(path + ")", n, open, close+1, res); }

6 易错要点
close < open 不是 close < n

右括号的数量受限于已放的左括号数，不是 n。只有 close < open 时才能放 )，否则会产生无效组合如 ())(。

字符串不可变

Python/Java 中字符串不可变，path + '(' 创建新串，无需手动撤销。但如果用 StringBuilder 或 list，需要手动 pop。

终止条件双等号

必须 open == close == n，不能只判断 len(path) == 2*n，因为剪枝已经保证合法性，但双等号更清晰。
