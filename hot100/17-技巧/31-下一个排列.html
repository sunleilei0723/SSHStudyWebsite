#31 下一个排列

给定一个整数数组 nums，找出 nums 的下一个排列。必须原地修改，只使用额外常数空间。如果不存在下一个排列，则将数组重新排列成最小的排列（升序）。

示例 1
输入：nums = [1,2,3]
输出：[1,3,2]
示例 2
输入：nums = [3,2,1]
输出：[1,2,3]
已经是最大排列，返回最小排列
示例 3
输入：nums = [1,1,5]
输出：[1,5,1]
2 题目拆解
当前排列 ➜ 字典序下一个 ➜ 原地修改 ➜ O(1) 空间

关键约束：

  • 找字典序下一个更大的排列
  • 原地修改，不能返回新数组
  • 如果是最大排列，返回最小排列
3 解题思路

三步法：

① 从右找第一个升序对 i ➜ ② 从右找第一个 > nums[i] 的 j ➜ ③ 交换 i,j，反转 i+1..end

直觉：要让排列"稍微变大"，需要从右往左找到第一个可以增大的位置，然后把后面部分变成最小的排列（反转即可）。

4 代码演示
def nextPermutation(nums): n = len(nums) # ① 从右找第一个升序对 i = n - 2 while i >= 0 and nums[i] >= nums[i+1]: i -= 1 if i >= 0: # 不是最大排列 # ② 从右找第一个 > nums[i] j = n - 1 while nums[j] <= nums[i]: j -= 1 nums[i], nums[j] = nums[j], nums[i] # ③ 交换 # ④ 反转 i+1..end nums[i+1:] = reversed(nums[i+1:])
public void nextPermutation(int[] nums) { int n = nums.length, i = n - 2; while (i >= 0 && nums[i] >= nums[i+1]) i--; if (i >= 0) { int j = n - 1; while (nums[j] <= nums[i]) j--; swap(nums, i, j); } reverse(nums, i+1, n-1); }
void nextPermutation(vector<int>& nums) { int n = nums.size(), i = n - 2; while (i >= 0 && nums[i] >= nums[i+1]) i--; if (i >= 0) { int j = n - 1; while (nums[j] <= nums[i]) j--; swap(nums[i], nums[j]); } reverse(nums.begin()+i+1, nums.end()); }

6 易错要点
找 i 的条件

找的是 nums[i] < nums[i+1]，不是 ≤。相等时不算升序，要继续往左找。

找 j 的条件

找的是 nums[j] > nums[i]，不是 ≥。如果相等，交换后没有变化。

i=-1 时也要反转

如果 i=-1（整个数组降序），不需要交换，但仍然要反转整个数组，变成升序。

7 扩展思考
  • 上一个排列：对称操作——找降序对，找更小值，交换后反转
  • 全排列：不断调用 nextPermutation 直到回到初始排列
  • 第 k 个排列：LeetCode 60，用数学方法直接计算
  • 含重复元素：nextPermutation 仍然适用，但全排列需要去重
