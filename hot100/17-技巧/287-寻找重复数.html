#287 寻找重复数

给定一个包含 n + 1 个整数的数组 nums，其数字都在 [1, n] 范围内，可知至少存在一个重复的整数。假设只有一个重复的整数，找出这个重复数。必须不修改数组，且只用常量级 O(1) 额外空间。

示例 1
输入：nums = [1,3,4,2,2]
输出：2
示例 2
输入：nums = [3,1,3,4,2]
输出：3
2 题目拆解
n+1 个数 ➜ 值域 [1,n] ➜ 找重复数 ➜ 不修改 O(1)

关键约束：

  • 不能修改数组（不能排序）
  • O(1) 额外空间（不能用哈希表）
  • 值域 [1,n]，长度 n+1，必有重复
3 解题思路

数组 → 链表映射：把 nums[i] 看成从节点 i 指向节点 nums[i] 的指针。重复数意味着有两个节点指向同一个节点 → 链表有环！

快慢指针找环入口：

slow = nums[slow] ➜ fast = nums[nums[fast]] ➜ 相遇点 ➜ 一个回起点，同步走 ➜ 再次相遇 = 环入口

为什么环入口就是重复数？因为环入口被两个不同的下标指向，即有两个 i 使得 nums[i] 相同。

4 代码演示
def findDuplicate(nums): slow = fast = 0 # 第一阶段：快慢指针找相遇点 while True: slow = nums[slow] fast = nums[nums[fast]] if slow == fast: break # 第二阶段：找环入口 slow = 0 while slow != fast: slow = nums[slow] fast = nums[fast] return slow
public int findDuplicate(int[] nums) { int slow = 0, fast = 0; do { slow = nums[slow]; fast = nums[nums[fast]]; } while (slow != fast); slow = 0; while (slow != fast) { slow = nums[slow]; fast = nums[fast]; } return slow; }
int findDuplicate(vector<int>& nums) { int slow = 0, fast = 0; do { slow = nums[slow]; fast = nums[nums[fast]]; } while (slow != fast); slow = 0; while (slow != fast) { slow = nums[slow]; fast = nums[fast]; } return slow; }

6 易错要点
起点是 0 不是 nums[0]

链表从虚拟头节点 0 开始，slow = nums[slow] 才是第一步移动。初始 slow = fast = 0。

第二阶段 slow 回到 0

相遇后，slow 要回到起点 0（不是回到 nums[0]），然后 slow 和 fast 同步走一步。

do-while vs while

初始 slow == fast == 0，如果用 while 循环会直接退出，所以第一阶段用 do-while。

7 扩展思考
  • 二分法：对值域 [1,n] 二分，统计 ≤ mid 的个数，O(n log n) 时间 O(1) 空间
  • 位运算法：统计每个 bit 位上 1 的个数，比无重复时多出的 bit 就是重复数的 bit
  • 如果可以修改数组：排序或标记法（把访问过的值取反），更简单
  • 多个重复数：快慢指针法不适用，需要其他方法
