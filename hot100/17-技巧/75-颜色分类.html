#75 颜色分类

给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。使用整数 0、1 和 2 分别表示红色、白色和蓝色。必须使用一趟扫描算法。

示例 1
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]
示例 2
输入：nums = [2,0,1]
输出：[0,1,2]
2 题目拆解
只有 0/1/2 ➜ 原地排序 ➜ 一趟扫描 ➜ 0..0..1..1..2..2

关键约束：

  • 只有三种值 0、1、2
  • 原地排序，不能用额外数组
  • 一趟扫描，O(n) 时间
3 解题思路

荷兰国旗问题 — 三指针分区：

  • low：0 的右边界（0 放在 [0, low-1]）
  • mid：当前扫描位置
  • high：2 的左边界（2 放在 [high+1, n-1]）
nums[mid]==0 ➜ swap(low,mid), low++, mid++
nums[mid]==1 ➜ mid++
nums[mid]==2 ➜ swap(mid,high), high--
4 代码演示
def sortColors(nums): low, mid, high = 0, 0, len(nums)-1 while mid <= high: if nums[mid] == 0: nums[low], nums[mid] = nums[mid], nums[low] low += 1; mid += 1 elif nums[mid] == 1: mid += 1 else: # nums[mid] == 2 nums[mid], nums[high] = nums[high], nums[mid] high -= 1
public void sortColors(int[] nums) { int low = 0, mid = 0, high = nums.length - 1; while (mid <= high) { if (nums[mid] == 0) { swap(nums, low++, mid++); } else if (nums[mid] == 1) { mid++; } else { swap(nums, mid, high--); } } }
void sortColors(vector<int>& nums) { int low = 0, mid = 0, high = nums.size() - 1; while (mid <= high) { if (nums[mid] == 0) swap(nums[low++], nums[mid++]); else if (nums[mid] == 1) mid++; else swap(nums[mid], nums[high--]); } }
5 题解过程
三指针分区过程
0 / 0
low mid high
点击 ▶ 开始三指针分区
6 易错要点
遇 2 不移动 mid

交换 nums[mid] 和 nums[high] 后，新的 nums[mid] 可能是 0，需要再判断，所以 mid 不能 ++。

遇 0 同时移动

交换 nums[low] 和 nums[mid] 后，low 和 mid 都要 ++。因为 low ≤ mid，交换来的值一定是 1。

循环条件

循环条件是 mid ≤ high，不是 mid < high。当 mid == high 时还需要判断一次。

7 扩展思考
  • 计数排序：统计 0/1/2 的个数再重写，两趟扫描但更直观
  • 四种颜色：需要更多指针或递归分区
  • 快速排序分区：荷兰国旗是快排三路分区的基础，处理大量重复元素
  • k 种颜色：一般化为多路分区问题
