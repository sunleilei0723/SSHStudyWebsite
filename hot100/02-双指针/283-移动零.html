#283 移动零

给定一个数组 nums，将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。必须原地操作，不能使用额外数组。

示例 1
输入：nums = [0,1,0,3,12]
输出：[1,3,12,0,0]
示例 2
输入：nums = [0]
输出：[0]
2 题目拆解
遍历数组 ➜ 遇非零前移 ➜ 剩余补零 ➜ 完成

关键约束：

  • 必须原地操作，O(1) 额外空间
  • 保持非零元素的相对顺序（稳定排序）
  • 最小化操作次数 — 非零元素最多被移动一次
3 解题思路

快慢指针法 O(n)：慢指针 slow 指向下一个非零元素应放的位置，快指针 fast 遍历数组。遇到非零就交换到 slow 位置，slow 右移。

fast 遍历 ➜ nums[fast]≠0? ➜ 交换到 slow 位置 ➜ slow++

本质：所有非零元素被依次"压缩"到数组前部，零自然被挤到末尾。

4 代码演示
def moveZeroes(nums): slow = 0 # 慢指针：下一个非零元素的位置 for fast in range(len(nums)): if nums[fast] != 0: # 遇到非零元素 nums[slow], nums[fast] = nums[fast], nums[slow] slow += 1 # 慢指针右移 # 无需显式补零，交换已将零移到末尾
public void moveZeroes(int[] nums) { int slow = 0; for (int fast = 0; fast < nums.length; fast++) { if (nums[fast] != 0) { int temp = nums[slow]; nums[slow] = nums[fast]; nums[fast] = temp; slow++; } } }
void moveZeroes(vector<int>& nums) { int slow = 0; for (int fast = 0; fast < nums.size(); fast++) { if (nums[fast] != 0) { swap(nums[slow], nums[fast]); slow++; } } }

6 易错要点
保持相对顺序

不能用"两头交换"的方式，那样会打乱非零元素的顺序。必须用快慢指针从左到右依次填充。

原地操作

题目要求 O(1) 空间，不能开新数组。快慢指针交换是原地操作，slow 之前的都是已就位的非零元素。

交换 vs 覆盖

可以直接 nums[slow]=nums[fast] 覆盖，最后再补零。但交换写法更简洁，且当 slow==fast 时交换自身无副作用。

7 扩展思考
  • 移动其他值：如果要把某个特定值移到末尾，同样的快慢指针框架，只需改判断条件
  • 稳定排序：本题本质是稳定的 partition 操作，类似快排的 partition 但要求稳定
  • 最少写操作：覆盖+补零比交换写操作更少（2n 次赋值 vs 最多 n 次交换）
  • 双端移动：如果不需要保持相对顺序，可以用两头指针，零移右、非零移左
