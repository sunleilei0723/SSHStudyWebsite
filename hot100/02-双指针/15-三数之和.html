#15 三数之和

给定整数数组 nums，找出所有和为 0 的三元组 [nums[i], nums[j], nums[k]]，且 i≠j≠k。结果中不能包含重复的三元组。

示例 1
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0，nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0
示例 2
输入：nums = [0,1,1]
输出：[]
2 题目拆解
排序数组 ➜ 固定一个数 ➜ 双指针找两数 ➜ 去重收集

关键约束：

  • 结果不能有重复三元组 — 排序后跳过相同值
  • 三元组内元素可以相同（如 [-1,-1,2]），但位置必须不同
  • 排序后，当 nums[i]>0 时可直接终止（后面不可能凑出0）
3 解题思路

暴力法 O(n³)：三重循环枚举 → 太慢，且去重麻烦

排序+双指针 O(n²)：先排序，固定第一个数 nums[i]，然后在 i+1..n-1 范围内用左右指针找两数之和等于 -nums[i]：

排序 ➜ 固定 i ➜ l=i+1, r=n-1 ➜ sum→0? 调指针 ➜ 去重

def threeSum(nums): nums.sort() res = [] for i in range(len(nums) - 2): if i > 0 and nums[i] == nums[i-1]: continue # 跳过重复i if nums[i] > 0: break # 优化：正数不可能凑0 l, r = i + 1, len(nums) - 1 while l < r: s = nums[i] + nums[l] + nums[r] if s < 0: l += 1 elif s > 0: r -= 1 else: res.append([nums[i], nums[l], nums[r]]) while l < r and nums[l] == nums[l+1]: l += 1 while l < r and nums[r] == nums[r-1]: r -= 1 l += 1; r -= 1 return res

6 易错要点
去重是关键

必须对 i、l、r 三个位置都去重。排序后，跳过与前一个相同的值。漏掉任何一个都会产生重复结果。

先排序

不排序就无法用双指针，也无法高效去重。排序是 O(n log n)，不影响整体 O(n²) 复杂度。

找到后双指针都要动

找到一个三元组后，l++ 和 r-- 要同时移动。只动一个，sum 不可能再为0（因为另一个没变）。

7 扩展思考
  • 四数之和：排序 + 两层循环固定两个数 + 双指针，O(n³)
  • K数之和：递归固定前 K-2 个数，最后两个用双指针
  • 最接近的三数之和：同样的框架，但记录与 target 最小的差值
  • 不排序方案：可以用哈希表，但去重更复杂，不如排序+双指针优雅
