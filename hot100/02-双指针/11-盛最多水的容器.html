#11 盛最多水的容器

给定 n 个非负整数 height[0..n-1]，每个数代表坐标中的一个点 (i, height[i])。找出两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。返回最大水量。

示例 1
输入：height = [1,8,6,2,5,4,8,3,7]
输出：49
图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]，在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49
示例 2
输入：height = [1,1]
输出：1
2 题目拆解
选两条线 ➜ 算面积 ➜ 找最大值 ➜ 返回结果

关键约束：

  • 面积 = min(height[l], height[r]) × (r - l)，由短板决定
  • 不能倾斜容器，水只能水平装
  • 至少需要两条线，n ≥ 2
3 解题思路

暴力法 O(n²)：枚举所有线对 → 太慢，淘汰

双指针贪心 O(n)：左右两端开始，每次移动较矮的指针。因为面积由短板决定，移动长板不可能增加面积，只有移动短板才可能找到更大的值：

l=0, r=n-1 ➜ 算面积，更新max ➜ 移动较矮的指针 ➜ l≥r 时结束
4 代码演示
def maxArea(height): l, r = 0, len(height) - 1 ans = 0 while l < r: area = min(height[l], height[r]) * (r - l) ans = max(ans, area) # 更新最大面积 if height[l] < height[r]: l += 1 # 移动较矮的指针 else: r -= 1 return ans
public int maxArea(int[] height) { int l = 0, r = height.length - 1, ans = 0; while (l < r) { int area = Math.min(height[l], height[r]) * (r - l); ans = Math.max(ans, area); if (height[l] < height[r]) l++; else r--; } return ans; }
int maxArea(vector<int>& height) { int l = 0, r = height.size() - 1, ans = 0; while (l < r) { int area = min(height[l], height[r]) * (r - l); ans = max(ans, area); if (height[l] < height[r]) l++; else r--; } return ans; }

6 易错要点
为什么移动短板

面积 = min(h[l],h[r]) × 宽度。移动长板：宽度变小，短板不变，面积一定变小。移动短板：宽度变小，但短板可能变高，面积可能变大。

等高时移哪边

height[l]==height[r] 时移哪边都行，因为两边都是短板。通常移左边，不影响最终结果。

不是找最大高度

最大面积不一定是两个最高柱子组成的！宽度也是关键因素。例如 [1,100,100,1]，最大面积是 100×2=200 而非 1×3=3。

7 扩展思考
  • 接雨水问题：同样用双指针，但需要跟踪左右最大高度，是本题的进阶版
  • 多解情况：如果要求输出所有最大面积对应的线对，需要额外记录
  • 三维版本：如果扩展到三维空间，问题变成 NP-hard，需要不同策略
  • 贪心正确性：可以用反证法证明，移动短板不会遗漏最优解
