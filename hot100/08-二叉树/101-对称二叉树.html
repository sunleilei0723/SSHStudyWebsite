#101 对称二叉树

给定一个二叉树，检查它是否是镜像对称的。即左右子树互为镜像：左子树的左 = 右子树的右，左子树的右 = 右子树的左。

示例 1
输入：root = [1,2,2,3,4,4,3]
输出：true
示例 2
输入：root = [1,2,2,null,3,null,3]
输出：false
2 题目拆解
输入二叉树 ➜ 比较左右子树 ➜ 左左 vs 右右 ➜ 左右 vs 右左

关键约束：

  • 对称 = 左右子树互为镜像
  • 不是左右子树相等，而是交叉相等
  • 根节点的左右子树要同时比较
3 解题思路

递归双指针：用两个指针分别从左右子树出发，交叉比较

left.val == right.val? ➜ check(left.left, right.right) ➜ check(left.right, right.left) ➜ 都对称→true
4 代码演示
def isSymmetric(root): if not root: return True return check(root.left, root.right) def check(left, right): if not left and not right: return True if not left or not right: return False return (left.val == right.val # 根相等 and check(left.left, right.right) # 左左 vs 右右 and check(left.right, right.left)) # 左右 vs 右左
public boolean isSymmetric(TreeNode root) { if (root == null) return true; return check(root.left, root.right); } private boolean check(TreeNode l, TreeNode r) { if (l == null && r == null) return true; if (l == null || r == null) return false; return l.val == r.val && check(l.left, r.right) && check(l.right, r.left); }
bool isSymmetric(TreeNode* root) { if (!root) return true; return check(root->left, root->right); } bool check(TreeNode* l, TreeNode* r) { if (!l && !r) return true; if (!l || !r) return false; return l->val == r->val && check(l->left, r->right) && check(l->right, r->left); }

6 易错要点
交叉比较

不是 left == right，而是 left.left vs right.right 和 left.right vs right.left，方向是交叉的！

终止条件

两个都为空 → true；一个为空一个不为空 → false；值不等 → false。顺序不能乱。

迭代写法

用队列实现：每次弹出两个节点比较，然后按交叉顺序入队 (l.left, r.right) 和 (l.right, r.left)。

7 扩展思考

  • 对称的子问题：整棵树对称 ⇔ 根的左右子树互为镜像
  • N叉树对称：子节点列表正反比较
