#108 将有序数组转换为二叉搜索树

给定一个升序数组 nums，将其转换为一棵高度平衡的二叉搜索树。平衡指每个节点左右子树高度差不超过1。

示例 1
输入：nums = [-10,-3,0,5,9]
输出：[0,-3,9,-10,null,5]
2 题目拆解
有序数组 ➜ 取中间元素为根 ➜ 左半递归建左子树 ➜ 右半递归建右子树

关键约束：

  • 数组已升序，天然满足BST中序遍历
  • 取中间元素为根可保证平衡
  • 分治：左半→左子树，右半→右子树
3 解题思路

分治递归：每次取中间元素为根，左半建左子树，右半建右子树

mid = (l+r)//2 ➜ root = nums[mid] ➜ build(l, mid-1) ➜ build(mid+1, r)
4 代码演示
def sortedArrayToBST(nums): def build(left, right): if left > right: return None mid = (left + right) // 2 # 取中间 root = TreeNode(nums[mid]) root.left = build(left, mid - 1) root.right = build(mid + 1, right) return root return build(0, len(nums) - 1)
public TreeNode sortedArrayToBST(int[] nums) { return build(nums, 0, nums.length - 1); } private TreeNode build(int[] nums, int l, int r) { if (l > r) return null; int mid = (l + r) / 2; TreeNode root = new TreeNode(nums[mid]); root.left = build(nums, l, mid - 1); root.right = build(nums, mid + 1, r); return root; }
TreeNode* sortedArrayToBST(vector<int>& nums) { return build(nums, 0, nums.size() - 1); } TreeNode* build(vector<int>& nums, int l, int r) { if (l > r) return nullptr; int mid = (l + r) / 2; TreeNode* root = new TreeNode(nums[mid]); root->left = build(nums, l, mid - 1); root->right = build(nums, mid + 1, r); return root; }

6 易错要点
mid的选择

mid = (l+r)//2 偏左，mid = (l+r+1)//2 偏右。两种都正确，只是构造的树形状不同。

边界条件

left > right 时返回 null，不是 left == right。当 left == right 时还要创建一个节点。

平衡的保证

每次取中间元素，左右子树节点数最多差1，天然保证高度平衡。

7 扩展思考
  • 有序链表转BST：LeetCode 109，用快慢指针找中点，或先转数组
  • 多种平衡BST：AVL树、红黑树都是平衡BST，但构造更复杂
  • 不是唯一解：题目允许多种合法答案，只要高度平衡即可
