#102 二叉树的层序遍历

给定二叉树 root，返回其节点值的层序遍历（即逐层地，从左到右访问所有节点）。

示例 1
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
2 题目拆解
根节点入队 ➜ 逐层出队 ➜ 孩子入队 ➜ 收集每层结果

关键约束：

  • 需要按层分组返回，不是简单的一维列表
  • 每层从左到右的顺序
  • BFS核心：用队列控制层次
3 解题思路

BFS队列法：每轮处理一整层，记录当前层节点数，依次出队并将孩子入队

queue=[root] ➜ size=queue.length ➜ 弹出size个节点 ➜ 孩子入队，收集本层
4 代码演示
def levelOrder(root): if not root: return [] res, queue = [], [root] while queue: size = len(queue) # 当前层节点数 level = [] for _ in range(size): node = queue.pop(0) level.append(node.val) if node.left: queue.append(node.left) if node.right: queue.append(node.right) res.append(level) return res
public List<List<Integer>> levelOrder(TreeNode root) { List<List<Integer>> res = new ArrayList<>(); if (root == null) return res; Queue<TreeNode> q = new LinkedList<>(); q.offer(root); while (!q.isEmpty()) { int size = q.size(); List<Integer> level = new ArrayList<>(); for (int i = 0; i < size; i++) { TreeNode node = q.poll(); level.add(node.val); if (node.left != null) q.offer(node.left); if (node.right != null) q.offer(node.right); } res.add(level); } return res; }
vector<vector<int>> levelOrder(TreeNode* root) { if (!root) return {}; vector<vector<int>> res; queue<TreeNode*> q; q.push(root); while (!q.empty()) { int size = q.size(); vector<int> level; for (int i = 0; i < size; i++) { TreeNode* node = q.front(); q.pop(); level.push_back(node->val); if (node->left) q.push(node->left); if (node->right) q.push(node->right); } res.push_back(level); } return res; }

6 易错要点
记录层大小

必须在循环前记录 size = queue.size()，否则循环中队列长度会变化，导致层次混乱。

空树判断

root == null 时直接返回空列表 []，不要忘记这个边界条件。

Python效率

Python 用 pop(0) 是 O(n)，建议用 collections.deque 的 popleft() 替代。
