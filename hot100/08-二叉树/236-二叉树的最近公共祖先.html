#236 二叉树的最近公共祖先

给定一个二叉树，找到两个指定节点 p 和 q 的最近公共祖先（LCA）。祖先的定义：节点x是节点p的祖先，当且仅当x是p的祖先节点或p本身。

示例 1
输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
输出：3
解释：节点 5 和节点 1 的最近公共祖先是节点 3
2 题目拆解
二叉树 + p + q ➜ DFS后序遍历 ➜ 左右子树找p/q ➜ 都找到→当前是LCA

关键约束：

  • LCA是离p和q最近的公共祖先
  • 一个节点可以是自己的祖先
  • p和q一定存在于树中
3 解题思路

DFS后序遍历：递归查找左右子树，三种情况：

左右都找到→当前是LCA ➜ 只左找到→返回左 ➜ 只右找到→返回右 ➜ 都没找到→返回null
4 代码演示
def lowestCommonAncestor(root, p, q): if not root or root == p or root == q: return root left = lowestCommonAncestor(root.left, p, q) right = lowestCommonAncestor(root.right, p, q) if left and right: # 左右都找到→当前是LCA return root return left if left else right # 返回非空的那个
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) { if (root == null || root == p || root == q) return root; TreeNode left = lowestCommonAncestor(root.left, p, q); TreeNode right = lowestCommonAncestor(root.right, p, q); if (left != null && right != null) return root; return left != null ? left : right; }
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) { if (!root || root == p || root == q) return root; TreeNode* left = lowestCommonAncestor(root->left, p, q); TreeNode* right = lowestCommonAncestor(root->right, p, q); if (left && right) return root; return left ? left : right; }

6 易错要点
终止条件

root==null || root==p || root==q 时返回root。找到p或q就立即向上返回，不再深入。

左右都非空

当left和right都非空，说明p和q分别在左右子树，当前节点就是LCA。这是核心判断！

一个为空

如果只有一个非空，说明p和q都在那一侧，返回非空的那个。它可能就是p或q本身。

7 扩展思考
  • BST的LCA：利用BST性质，值在p和q之间的就是LCA
  • 多次查询：Tarjan离线算法或倍增法，预处理后O(1)查询
  • 节点不保证存在：需要额外判断p和q是否真的在树中
