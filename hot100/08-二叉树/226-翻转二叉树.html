#226 翻转二叉树

给定一棵二叉树的根节点 root，翻转这棵二叉树，并返回其根节点。即交换每个节点的左右子树。

2 题目拆解
输入二叉树 ➜ 交换每个节点左右子树 ➜ 递归处理子树 ➜ 返回翻转后的根

关键约束：

  • 翻转 = 交换每个节点的 left 和 right 指针
  • 是原地修改，不需要创建新树
  • 著名的"Homebrew作者Google面试题" 😄
3 解题思路

DFS递归：对每个节点，先翻转左右子树，再交换左右指针

BFS迭代：用队列层序遍历，每弹出一个节点就交换其左右孩子

访问节点 ➜ 递归翻转左子树 ➜ 递归翻转右子树 ➜ 交换left和right
4 代码演示
def invertTree(root): if not root: return None root.left, root.right = root.right, root.left # 交换 invertTree(root.left) # 翻转左子树 invertTree(root.right) # 翻转右子树 return root
public TreeNode invertTree(TreeNode root) { if (root == null) return null; TreeNode temp = root.left; root.left = root.right; root.right = temp; // 交换 invertTree(root.left); invertTree(root.right); return root; }
TreeNode* invertTree(TreeNode* root) { if (!root) return nullptr; swap(root->left, root->right); // 交换 invertTree(root->left); invertTree(root->right); return root; }

6 易错要点
交换顺序

先交换再递归，或先递归再交换都可以。但必须确保 left 和 right 同时赋值，避免覆盖丢失。

原地修改

交换的是指针/引用，不是创建新节点。Python 可以 a,b = b,a，Java 需要临时变量。

空节点处理

递归终止条件是 root == null，叶子节点的左右孩子为 null 也会被交换（虽然无效果）。

7 扩展思考
  • 对称判断：翻转后与原树相同 ⇔ 原树对称（LeetCode 101）
  • N叉树翻转：反转子节点列表即可，children.reverse()
  • 迭代BFS写法：用队列层序遍历，每弹出一个节点交换其左右孩子
  • 应用场景：镜像操作、图像翻转等
