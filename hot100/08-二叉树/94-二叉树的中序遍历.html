#94 二叉树的中序遍历

给定一个二叉树的根节点 root，返回它的 中序遍历 结果。中序遍历的顺序为：左子树 → 根节点 → 右子树。

示例 1
输入：root = [1,null,2,3]
输出：[1,3,2]
2 题目拆解
输入二叉树 ➜ 左→根→右遍历 ➜ 收集节点值 ➜ 返回遍历序列

关键约束：

  • 中序遍历顺序：左子树 → 根 → 右子树
  • 可以用递归或迭代（栈）两种方式实现
  • 迭代法需要用栈模拟递归调用过程
3 解题思路

递归法：最直观，按"左→根→右"顺序递归访问

迭代法（栈）：用栈模拟递归，一路向左压栈，弹出时访问，再转向右子树：

从根出发一路向左压栈 ➜ 栈顶弹出→访问 ➜ 转向右子树 ➜ 重复直到栈空且无右子树
4 代码演示
def inorderTraversal(root): res = [] def dfs(node): if not node: return dfs(node.left) # 左 res.append(node.val) # 根 dfs(node.right) # 右 dfs(root) return res # 迭代法（栈） def inorderIterative(root): res, stack = [], [] curr = root while curr or stack: while curr: # 一路向左压栈 stack.append(curr) curr = curr.left curr = stack.pop() # 弹出访问 res.append(curr.val) curr = curr.right # 转右子树 return res
public List<Integer> inorderTraversal(TreeNode root) { List<Integer> res = new ArrayList<>(); dfs(root, res); return res; } private void dfs(TreeNode node, List<Integer> res) { if (node == null) return; dfs(node.left, res); // 左 res.add(node.val); // 根 dfs(node.right, res); // 右 }

6 易错要点
遍历顺序

中序 = 左→根→右，不要和前序（根左右）、后序（左右根）搞混。口诀："中"序就是根在"中间"。

迭代法栈操作

先一路向左压栈到底，弹出时才访问值，然后转向右子树。不要在压栈时就访问！

BST的中序特性

对二叉搜索树做中序遍历，结果一定是递增序列，这是BST最重要的性质之一。

7 扩展思考
  • Morris遍历：O(1)空间的中序遍历，利用线索二叉树思想，修改树结构临时指向
  • 前序/后序迭代：前序最简单（先访问再压栈），后序需要双栈或标记法
  • 验证BST：利用中序遍历递增性质，只需检查相邻元素是否递增
  • 第K小元素：中序遍历到第K个即可停止，无需完整遍历
