#105 从前序与中序遍历序列构造二叉树

给定两个整数数组 preorder 和 inorder，构造二叉树并返回其根节点。前序确定根，中序划分左右子树。

示例 1
输入：preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出：[3,9,20,null,null,15,7]
2 题目拆解
preorder + inorder ➜ 前序首元素=根 ➜ 中序找根划分左右 ➜ 递归建子树

关键约束：

  • 前序第一个元素 = 根节点
  • 中序中根的位置划分左子树和右子树
  • 递归对左右子树重复上述过程
3 解题思路

分治递归：前序首元素为根，在中序中找到根的位置，左边是左子树，右边是右子树

pre[0]=根 ➜ in中找根位置idx ➜ in[0:idx]=左子树 ➜ in[idx+1:]=右子树
4 代码演示
def buildTree(preorder, inorder): idx_map = {v: i for i, v in enumerate(inorder)} def build(pre_l, pre_r, in_l, in_r): if pre_l > pre_r: return None root_val = preorder[pre_l] # 前序首元素=根 root = TreeNode(root_val) idx = idx_map[root_val] # 中序中根的位置 left_size = idx - in_l # 左子树大小 root.left = build(pre_l+1, pre_l+left_size, in_l, idx-1) root.right = build(pre_l+left_size+1, pre_r, idx+1, in_r) return root return build(0, len(preorder)-1, 0, len(inorder)-1)
private Map<Integer, Integer> idxMap; private int[] preorder; public TreeNode buildTree(int[] preorder, int[] inorder) { this.preorder = preorder; idxMap = new HashMap<>(); for (int i = 0; i < inorder.length; i++) idxMap.put(inorder[i], i); return build(0, preorder.length - 1, 0, inorder.length - 1); } TreeNode build(int pl, int pr, int il, int ir) { if (pl > pr) return null; int rootVal = preorder[pl]; int idx = idxMap.get(rootVal); int leftSize = idx - il; TreeNode root = new TreeNode(rootVal); root.left = build(pl+1, pl+leftSize, il, idx-1); root.right = build(pl+leftSize+1, pr, idx+1, ir); return root; }
unordered_map<int,int> idxMap; vector<int> pre; TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) { pre = preorder; for(int i=0;i<inorder.size();i++) idxMap[inorder[i]]=i; return build(0,pre.size()-1,0,inorder.size()-1); } TreeNode* build(int pl,int pr,int il,int ir){ if(pl>pr)return nullptr; int rv=pre[pl], idx=idxMap[rv], ls=idx-il; TreeNode* root=new TreeNode(rv); root->left=build(pl+1,pl+ls,il,idx-1); root->right=build(pl+ls+1,pr,idx+1,ir); return root; }

6 易错要点
下标计算

左子树前序范围 [pl+1, pl+leftSize]，右子树 [pl+leftSize+1, pr]。leftSize = idx - in_l，不是 idx！

哈希表优化

用哈希表预处理中序值→下标映射，避免每次线性查找根位置，O(n)→O(1)。

元素不重复

题目保证无重复元素，所以哈希映射是可行的。如果有重复，需要额外处理。
