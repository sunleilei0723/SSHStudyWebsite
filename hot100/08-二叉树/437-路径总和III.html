#437 路径总和III

给定一个二叉树的根节点 root 和整数 targetSum，返回路径节点值之和等于 targetSum 的路径数目。路径不需要从根开始，也不需要在叶子结束，但必须向下。

2 题目拆解
二叉树 + targetSum ➜ 前缀和+哈希 ➜ 查找prefix-target ➜ 回溯恢复

关键约束：

  • 路径任意起点，不必从根开始
  • 路径必须向下（父→子方向）
  • 前缀和思路：若 prefix[j] - prefix[i] = target，则 i→j 路径和为 target
3 解题思路

前缀和+哈希表：DFS遍历时维护从根到当前节点的前缀和，查哈希表中是否有 prefix-target

DFS遍历 ➜ 累加前缀和 ➜ 查hash[prefix-target] ➜ 回溯时删前缀和
4 代码演示
def pathSum(root, targetSum): prefix = defaultdict(int) prefix[0] = 1 # 前缀和0出现1次 def dfs(node, curr_sum): if not node: return 0 curr_sum += node.val count = prefix[curr_sum - targetSum] # 查匹配 prefix[curr_sum] += 1 count += dfs(node.left, curr_sum) count += dfs(node.right, curr_sum) prefix[curr_sum] -= 1 # 回溯！ return count return dfs(root, 0)
private Map<Long, Integer> prefix = new HashMap<>(); private int target, ans; public int pathSum(TreeNode root, int targetSum) { prefix.put(0L, 1); target = targetSum; ans = 0; dfs(root, 0L); return ans; } private void dfs(TreeNode node, long sum) { if (node == null) return; sum += node.val; ans += prefix.getOrDefault(sum - target, 0); prefix.merge(sum, 1, Integer::sum); dfs(node.left, sum); dfs(node.right, sum); prefix.merge(sum, -1, Integer::sum); // 回溯 }
unordered_map<long long,int> prefix; int target, ans=0; int pathSum(TreeNode* root, int t) { prefix[0]=1; target=t; dfs(root,0); return ans; } void dfs(TreeNode* node, long long sum) { if(!node)return; sum += node->val; ans += prefix[sum - target]; prefix[sum]++; dfs(node->left, sum); dfs(node->right, sum); prefix[sum]--; // 回溯 }

6 易错要点
必须回溯

DFS返回时必须 prefix[curr]--，否则左右子树的前缀和会互相干扰！这是最容易忘的。

prefix[0]=1

初始化 prefix[0]=1，表示前缀和恰好等于target时（从根到当前），也算一条路径。

溢出问题

前缀和可能超出int范围，Java/C++用 long/long long，Python自动处理大整数。

7 扩展思考
  • 暴力法O(n²)：每个节点作为起点DFS，双重递归
  • 前缀和O(n)：类似数组的和为K的子数组，迁移到树上
  • 路径总和I/II：I要求根到叶子，II要求返回所有路径
