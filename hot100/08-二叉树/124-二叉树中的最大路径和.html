#124 二叉树中的最大路径和

给定一个二叉树，返回其最大路径和。路径被定义为从任意节点出发，沿父→子连接到达任意节点的序列。同一个节点在路径中最多出现一次。

示例 1
输入：root = [-10,9,20,null,null,15,7]
输出：42
解释：最优路径是 15→20→7
2 题目拆解
输入二叉树 ➜ DFS求单侧最大贡献 ➜ 更新全局最大路径 ➜ 返回max(left,right)+val

关键约束：

  • 路径任意起点和终点，不一定经过根
  • 路径只能走父→子方向，不能回头
  • 节点值可能为负数，负贡献要舍弃
3 解题思路

DFS后序遍历：对每个节点，计算"单侧最大贡献"（只能选左或右一侧），同时用"左+根+右"更新全局最大

递归求左贡献 ➜ 递归求右贡献 ➜ ans=max(ans, left+right+val) ➜ return max(left,right)+val
4 代码演示
def maxPathSum(root): ans = float('-inf') def dfs(node): nonlocal ans if not node: return 0 left = max(dfs(node.left), 0) # 负贡献舍弃 right = max(dfs(node.right), 0) # 负贡献舍弃 ans = max(ans, left + right + node.val) # 更新全局 return max(left, right) + node.val # 单侧贡献 dfs(root) return ans
private int ans = Integer.MIN_VALUE; public int maxPathSum(TreeNode root) { dfs(root); return ans; } private int dfs(TreeNode node) { if (node == null) return 0; int left = Math.max(dfs(node.left), 0); int right = Math.max(dfs(node.right), 0); ans = Math.max(ans, left + right + node.val); return Math.max(left, right) + node.val; }
int ans = INT_MIN; int maxPathSum(TreeNode* root) { dfs(root); return ans; } int dfs(TreeNode* node) { if (!node) return 0; int left = max(dfs(node->left), 0); int right = max(dfs(node->right), 0); ans = max(ans, left + right + node->val); return max(left, right) + node->val; }

6 易错要点
负贡献舍弃

max(dfs(left), 0)：如果子树贡献为负，不如不走，取0。这是本题的关键！

返回值 vs 全局最大

返回给父节点的是单侧贡献（只能选左或右），全局最大可能是双侧（左+根+右）。两者不同！

初始值

ans 初始化为 INT_MIN 或 -inf，不能是0，因为所有节点可能都是负数。

7 扩展思考
  • 与#543直径对比：思路完全一致！543求的是边数（深度之和），本题求的是节点值之和
  • 路径必须连续：路径中不能跳过节点，必须沿父子连接连续走
  • 输出路径：如果要输出具体路径，需要额外记录路径信息
