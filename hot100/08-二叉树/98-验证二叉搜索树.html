#98 验证二叉搜索树

给定一个二叉树，判断其是否是一个有效的二叉搜索树。BST定义：左子树所有节点值 < 根 < 右子树所有节点值，且左右子树也是BST。

2 题目拆解
输入二叉树 ➜ 中序遍历 ➜ 检查递增性 ➜ 返回是否BST

关键约束：

  • BST要求左 < 根 < 右（严格小于，不能等于）
  • 不仅是直接子节点，而是整棵子树的所有节点
  • 中序遍历BST结果一定是严格递增序列
3 解题思路

方法1：中序遍历 — 中序遍历结果是否严格递增

方法2：递归带范围 — 每个节点必须在 (lower, upper) 范围内

root在(lower,upper)? ➜ 左子树→upper=root.val ➜ 右子树→lower=root.val ➜ 都满足→true

public boolean isValidBST(TreeNode root) { return check(root, Long.MIN_VALUE, Long.MAX_VALUE); } private boolean check(TreeNode node, long lower, long upper) { if (node == null) return true; if (node.val <= lower || node.val >= upper) return false; return check(node.left, lower, node.val) && check(node.right, node.val, upper); }

6 易错要点
不仅是直接子节点

树 [5,1,4,null,null,3,6] 中，3 < 4 但 3 < 5 不满足！右子树所有值必须 > 根，不仅是直接右孩子。

用long防溢出

Java/C++ 中节点值可能是 INT_MIN 或 INT_MAX，初始边界要用 Long.MIN_VALUE 避免溢出。

严格小于

BST定义是严格小于/大于，不允许等于。node.val <= lower 就返回 false。

7 扩展思考
  • 中序遍历法：维护一个 prev 指针，每次检查当前 > prev
  • 恢复BST：LeetCode 99，中序遍历找到两个逆序位置并交换
  • BST的删除/插入：验证是基础操作，理解BST性质很重要
