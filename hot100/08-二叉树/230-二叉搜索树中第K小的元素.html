#230 二叉搜索树中第K小的元素

给定一个二叉搜索树 root 和一个整数 k，返回BST中第 k 小 的元素值。BST中序遍历结果为递增序列。

示例 1
输入：root = [3,1,4,null,2], k = 1
输出：1
2 题目拆解
BST + k ➜ 中序遍历 ➜ 计数到第k个 ➜ 返回该节点值

关键约束：

  • BST中序遍历 = 递增序列
  • 第k小 = 中序遍历第k个访问的节点
  • 可以提前终止：找到第k个就停止
3 解题思路

中序遍历+计数：中序遍历BST，每访问一个节点计数+1，到k时返回

中序遍历 ➜ 访问节点时cnt++ ➜ cnt==k → 返回 ➜ 否则继续遍历
4 代码演示
def kthSmallest(root, k): cnt = 0 res = None def inorder(node): nonlocal cnt, res if not node or res is not None: return inorder(node.left) # 左 cnt += 1 if cnt == k: # 第k个！ res = node.val return inorder(node.right) # 右 inorder(root) return res
private int cnt = 0, res = 0; public int kthSmallest(TreeNode root, int k) { inorder(root, k); return res; } private void inorder(TreeNode node, int k) { if (node == null || cnt >= k) return; inorder(node.left, k); if (++cnt == k) { res = node.val; return; } inorder(node.right, k); }
int cnt = 0, res = 0; int kthSmallest(TreeNode* root, int k) { inorder(root, k); return res; } void inorder(TreeNode* node, int k) { if (!node || cnt >= k) return; inorder(node->left, k); if (++cnt == k) { res = node->val; return; } inorder(node->right, k); }

6 易错要点
提前终止

找到第k个后立即返回，不需要继续遍历。用全局变量或引用传递结果，避免无效递归。

迭代法更直观

用栈迭代中序遍历，每弹出一个节点 cnt++，到k时直接返回，不需要全局变量。

Follow up

如果频繁查询，可以在节点中维护左子树大小，实现O(h)查询，类似平衡树的rank操作。

7 扩展思考
  • 第K大元素：逆中序遍历（右→根→左），或第(n-k+1)小
  • 优化查询：在节点中存左子树节点数，实现O(h)的rank查询
  • 数组第K大：快速选择算法，平均O(n)
