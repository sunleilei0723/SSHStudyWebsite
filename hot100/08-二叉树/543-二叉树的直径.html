#543 二叉树的直径

给定一棵二叉树，返回其直径长度——即任意两节点间最长路径的边数。该路径可能经过也可能不经过根节点。

示例 1
输入：root = [1,2,3,4,5]
输出：3
解释：3是路径长度 [4,2,1,3] 或者 [5,2,1,3]
2 题目拆解
输入二叉树 ➜ DFS求每个节点深度 ➜ 更新最大直径 ➜ 返回直径

关键约束：

  • 直径 = 左深度 + 右深度（经过当前节点的最长路径边数）
  • 路径可能不经过根节点，所以每个节点都要检查
  • 返回给父节点的是 max(left, right) + 1（单侧深度）
3 解题思路

DFS后序遍历：求深度的同时，顺便计算经过当前节点的直径

递归求左深度 ➜ 递归求右深度 ➜ diameter = left+right ➜ return max(left,right)+1
4 代码演示
def diameterOfBinaryTree(root): ans = 0 def depth(node): nonlocal ans if not node: return 0 l = depth(node.left) r = depth(node.right) ans = max(ans, l + r) # 更新直径 return max(l, r) + 1 # 返回深度 depth(root) return ans
private int ans = 0; public int diameterOfBinaryTree(TreeNode root) { depth(root); return ans; } private int depth(TreeNode node) { if (node == null) return 0; int l = depth(node.left); int r = depth(node.right); ans = Math.max(ans, l + r); return Math.max(l, r) + 1; }

6 易错要点
直径是边数

直径 = left_depth + right_depth，不是节点数！如果路径有3个节点，直径（边数）=2。

不一定过根

最长路径可能不经过根节点！所以每个节点都要用 l+r 更新全局最大值。

返回值 vs 直径

返回给父节点的是 max(l,r)+1（单侧深度），不是 l+r（直径）。两者要分清！
