#104 二叉树的最大深度

给定一个二叉树 root，返回其最大深度。二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

示例 1
输入：root = [3,9,20,null,null,15,7]
输出：3
2 题目拆解
输入二叉树 ➜ DFS递归求深度 ➜ max(左深, 右深)+1 ➜ 返回最大深度

关键约束：

  • 深度 = 根到最远叶子的节点数（不是边数）
  • 空树深度为 0，只有根节点深度为 1
  • 核心递推：depth = max(left_depth, right_depth) + 1
3 解题思路

DFS递归（后序遍历）：先递归求左右子树深度，再取较大值+1

BFS层序遍历：逐层遍历，记录层数即为深度

递归到叶子 ➜ 空节点返回0 ➜ 左右取max+1 ➜ 逐层向上返回
4 代码演示
def maxDepth(root): if not root: return 0 # 空节点深度为0 left = maxDepth(root.left) # 左子树深度 right = maxDepth(root.right) # 右子树深度 return max(left, right) + 1 # 取较大值+1
public int maxDepth(TreeNode root) { if (root == null) { return 0; // 空节点深度为0 } int left = maxDepth(root.left); int right = maxDepth(root.right); return Math.max(left, right) + 1; }
int maxDepth(TreeNode* root) { if (!root) { return 0; // 空节点深度为0 } int left = maxDepth(root->left); int right = maxDepth(root->right); return max(left, right) + 1; }

6 易错要点
深度 vs 高度

本题求的是深度（根到叶），空节点返回 0。高度定义可能不同，注意题目说的是"节点数"不是"边数"。

递归终止条件

空节点返回 0，不是 -1 或 1。只有根节点时 max(0,0)+1=1，正确。

BFS替代方案

也可以用层序遍历统计层数，每遍历完一层 depth++，但DFS递归更简洁。
