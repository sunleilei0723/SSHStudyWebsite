#114 二叉树展开为链表

给定一个二叉树，原地将它展开为一个单链表。展开后的链表与二叉树前序遍历顺序相同，每个节点的右子指针指向下一个节点，左子指针为 null。

示例 1
输入：root = [1,2,5,3,4,null,6]
输出：[1,null,2,null,3,null,4,null,5,null,6]
2 题目拆解
输入二叉树 ➜ 前序遍历顺序 ➜ 左→null, 右→next ➜ 原地展开

关键约束：

  • 展开顺序 = 前序遍历（根→左→右）
  • 原地修改，不能新建节点
  • 左指针设为null，右指针指向下一个前序节点
3 解题思路

方法1：前序遍历+数组 — 先前序遍历存节点，再依次修改指针

方法2：递归后序 — 先展开左右子树，再拼接：左子树接右，原右子树接左子树末尾

方法3：迭代（最优） — 对每个节点，把左子树插入到右子树前面

找到左子树最右节点 ➜ 原右子树接到最右 ➜ 左子树移到右边 ➜ root = root.right
4 代码演示
def flatten(root): curr = root while curr: if curr.left: # 找左子树最右节点 prev = curr.left while prev.right: prev = prev.right prev.right = curr.right # 原右子树接到最右 curr.right = curr.left # 左移到右 curr.left = None # 左置空 curr = curr.right
public void flatten(TreeNode root) { TreeNode curr = root; while (curr != null) { if (curr.left != null) { TreeNode prev = curr.left; while (prev.right != null) prev = prev.right; prev.right = curr.right; curr.right = curr.left; curr.left = null; } curr = curr.right; } }
void flatten(TreeNode* root) { TreeNode* curr = root; while (curr) { if (curr->left) { TreeNode* prev = curr->left; while (prev->right) prev = prev->right; prev->right = curr->right; curr->right = curr->left; curr->left = nullptr; } curr = curr->right; } }

6 易错要点
左子树最右节点

关键步骤：找到左子树的最右节点，把原右子树接上去。否则原右子树会丢失！

顺序很重要

必须先接右子树到最右，再移动左到右，再置空左。顺序反了会丢失子树。

原地修改

题目要求原地展开，不能返回新链表。所有操作都在原树上修改指针完成。

7 扩展思考
  • 递归后序写法：先展开左右子树，再拼接。需要记录左子树末尾
  • 前序遍历+数组：最简单但O(n)空间，先存节点再改指针
  • 类似Morris遍历：迭代法的思路与Morris遍历非常相似
