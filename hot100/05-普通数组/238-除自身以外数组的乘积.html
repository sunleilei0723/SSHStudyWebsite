#238 除自身以外数组的乘积

给你一个整数数组 nums，返回数组 answer，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。题目数据保证数组之中任意元素的全部前缀元素和后缀的乘积都在 32 位整数范围内。请不要使用除法，且在 O(n) 时间复杂度内完成。

示例 1
输入：nums = [1,2,3,4]
输出：[24,12,8,6]
示例 2
输入：nums = [-1,1,0,-3,3]
输出：[0,0,9,0,0]
2 题目拆解
输入数组 ➜ 算左前缀积 ➜ 算右前缀积 ➜ 左×右=答案

关键约束：

  • 不能使用除法，即使算出总乘积再除也不行
  • O(n) 时间，且除了输出数组外只用 O(1) 额外空间
  • answer[i] = 左边所有数的积 × 右边所有数的积
3 解题思路

暴力法 O(n²)：对每个位置，遍历其他元素求积 → 太慢

左右前缀积 O(n)：核心思想：answer[i] = left[i] × right[i]，其中 left[i] 是 i 左边所有数的积，right[i] 是 i 右边所有数的积。优化：用输出数组存 left，再用一个变量从右往左累乘 right：

从左到右算 left 前缀积 ➜ 从右到左累乘 right ➜ left[i] × right → answer[i]
4 代码演示
def productExceptSelf(nums): n = len(nums) ans = [1] * n # 第1遍：从左到右，算左前缀积 left = 1 for i in range(n): ans[i] = left left *= nums[i] # 第2遍：从右到左，算右前缀积 right = 1 for i in range(n-1, -1, -1): ans[i] *= right right *= nums[i] return ans
public int[] productExceptSelf(int[] nums) { int n = nums.length; int[] ans = new int[n]; int left = 1; for (int i = 0; i < n; i++) { ans[i] = left; left *= nums[i]; } int right = 1; for (int i = n-1; i >= 0; i--) { ans[i] *= right; right *= nums[i]; } return ans; }
vector<int> productExceptSelf(vector<int>& nums) { int n = nums.size(); vector<int> ans(n, 1); int left = 1; for (int i = 0; i < n; i++) { ans[i] = left; left *= nums[i]; } int right = 1; for (int i = n-1; i >= 0; i--) { ans[i] *= right; right *= nums[i]; } return ans; }

6 易错要点
不能用除法

即使算出总乘积再除以 nums[i] 也不行。如果有0存在，除法会出错；题目也明确禁止。

两遍遍历顺序

第1遍从左到右算左前缀积，第2遍从右到左算右前缀积。方向不能搞反，否则结果错误。

初始值为1

left 和 right 初始值必须为1，不是0。1是乘法的单位元，0会使得所有乘积都变成0。

7 扩展思考
  • 如果允许除法：先算总乘积，再逐个除。但需处理0的情况：0个、1个、多个0分别讨论
  • 如果要求 O(1) 空间：输出数组不算额外空间，本解法已满足
  • 如果数组含0：本解法天然支持，0的位置 answer 为其他元素乘积，非0位置若有两个以上0则为0
  • 前缀和 vs 前缀积：思想类似，一个是加法累积，一个是乘法累积
