#189 轮转数组

给定一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。要求原地修改数组，O(1) 额外空间。

示例 1
输入：nums = [1,2,3,4,5,6,7], k = 3
输出：[5,6,7,1,2,3,4]
向右轮转 1 步: [7,1,2,3,4,5,6] → 向右轮转 2 步: [6,7,1,2,3,4,5] → 向右轮转 3 步: [5,6,7,1,2,3,4]
示例 2
输入：nums = [-1,-100,3,99], k = 2
输出：[3,99,-1,-100]
2 题目拆解
输入数组 ➜ k 取模 ➜ 三次反转 ➜ 完成轮转

关键约束：

  • 必须原地修改，O(1) 额外空间
  • k 可能大于数组长度，需先 k %= n
  • 向右轮转 k 等价于：后 k 个元素移到前面
3 解题思路

额外数组法 O(n) 空间：开辟新数组，ans[(i+k)%n] = nums[i] → 不满足 O(1) 空间

三次反转法 O(1) 空间：巧妙利用反转操作。向右轮转 k = 将后 k 个移到前面，等价于：

反转整个数组 ➜ 反转前 k 个 ➜ 反转后 n-k 个

例：[1,2,3,4,5,6,7] k=3 → 反转全部 [7,6,5,4,3,2,1] → 反转前3 [5,6,7,4,3,2,1] → 反转后4 [5,6,7,1,2,3,4]

4 代码演示
def rotate(nums, k): n = len(nums) k %= n # k 可能大于 n nums.reverse() # 第1次：反转全部 nums[:k] = reversed(nums[:k]) # 第2次：反转前k个 nums[k:] = reversed(nums[k:]) # 第3次：反转后n-k个
public void rotate(int[] nums, int k) { int n = nums.length; k %= n; reverse(nums, 0, n - 1); // 反转全部 reverse(nums, 0, k - 1); // 反转前k个 reverse(nums, k, n - 1); // 反转后n-k个 } void reverse(int[] nums, int l, int r) { while (l < r) { int t = nums[l]; nums[l++] = nums[r]; nums[r--] = t; } }
void rotate(vector<int>& nums, int k) { int n = nums.size(); k %= n; reverse(nums.begin(), nums.end()); // 反转全部 reverse(nums.begin(), nums.begin() + k); // 反转前k个 reverse(nums.begin() + k, nums.end()); // 反转后n-k个 }

6 易错要点
k 必须取模

k %= n 不能忘！当 k > n 时，轮转 k 次等价于轮转 k % n 次。

反转顺序

三次反转的顺序是：全部 → 前k → 后n-k。顺序不能乱，否则结果错误。

原地修改

题目要求原地修改，不能返回新数组。Python 切片赋值 nums[:]= 是原地修改，但 nums= 不是。

7 扩展思考
  • 环状替换法：O(1) 空间的另一种解法，从位置0开始，把元素放到它最终位置，直到回到起点
  • 向左轮转：等价于向右轮转 n-k 次，或反转顺序改为 前n-k → 后k → 全部
  • 字符串轮转：LeetCode 796，判断一个字符串是否由另一个旋转得到
  • 寻找旋转排序数组最小值：LeetCode 153，二分查找的应用
