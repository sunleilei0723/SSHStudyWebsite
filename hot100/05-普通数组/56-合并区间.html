#56 合并区间

以数组 intervals 表示若干个区间的集合，请你合并所有重叠的区间，并返回一个不重叠的区间数组。

示例 1
输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
输出：[[1,6],[8,10],[15,18]]
区间 [1,3] 和 [2,6] 重叠, 合并为 [1,6]
示例 2
输入：intervals = [[1,4],[4,5]]
输出：[[1,5]]
2 题目拆解
输入区间 ➜ 按左端点排序 ➜ 逐个合并 ➜ 返回结果

关键约束：

  • 区间 [a,b] 和 [c,d] 重叠当且仅当 c ≤ b
  • 合并后区间 [a, max(b,d)]
  • 输入区间不一定有序，必须先排序
3 解题思路

排序 + 贪心 O(n log n)：先按区间左端点排序，然后遍历：如果当前区间与结果中最后一个区间重叠，就合并（更新右端点为较大值）；否则直接加入结果。

按左端点排序 ➜ 取第一个区间入结果 ➜ 遍历：重叠→合并，不重叠→加入 ➜ 返回结果
4 代码演示
def merge(intervals): intervals.sort(key=lambda x: x[0]) # 按左端点排序 merged = [] for interval in intervals: if not merged or merged[-1][1] < interval[0]: merged.append(interval) # 不重叠，直接加入 else: merged[-1][1] = max(merged[-1][1], interval[1]) # 合并 return merged
public int[][] merge(int[][] intervals) { Arrays.sort(intervals, (a, b) -> a[0] - b[0]); List<int[]> merged = new ArrayList<>(); for (int[] iv : intervals) { if (merged.isEmpty() || merged.get(merged.size()-1)[1] < iv[0]) merged.add(iv); else merged.get(merged.size()-1)[1] = Math.max(merged.get(merged.size()-1)[1], iv[1]); } return merged.toArray(new int[merged.size()][]); }
vector<vector<int>> merge(vector<vector<int>>& intervals) { sort(intervals.begin(), intervals.end()); vector<vector<int>> merged; for (auto& iv : intervals) { if (merged.empty() || merged.back()[1] < iv[0]) merged.push_back(iv); else merged.back()[1] = max(merged.back()[1], iv[1]); } return merged; }

6 易错要点
必须先排序

如果不排序，[1,4] 和 [2,3] 可能不相邻，无法正确合并。排序保证相邻区间才可能重叠。

重叠条件

判断重叠用 merged[-1][1] >= interval[0]，注意是 ≥ 不是 >。如 [1,4] 和 [4,5] 也算重叠。

合并取max

合并时右端点取 max，不是直接用新区间的右端点。如 [1,6] 和 [2,4] 合并后是 [1,6]。
