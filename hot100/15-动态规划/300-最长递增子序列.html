#300 最长递增子序列

给你一个整数数组 nums，找到其中最长严格递增子序列的长度。子序列是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。

示例 1
输入：nums = [10,9,2,5,3,7,101,18]
输出：4
最长递增子序列是 [2,3,7,101]，长度为 4
示例 2
输入：nums = [0,1,0,3,2,3]
输出：4
2 题目拆解
整数数组 ➜ 严格递增 ➜ 子序列（可不连续） ➜ 最长长度

关键约束：

  • 严格递增：相邻元素不能相等
  • 子序列：可以跳过元素，不要求连续
  • 只求长度，不需要具体序列
3 解题思路

DP O(n²)：定义 dp[i] = 以 nums[i] 结尾的 LIS 长度：

dp[i] = 1（自身） ➜ 若 nums[j] < nums[i]：dp[i] = max(dp[j]+1) ➜ 答案 = max(dp)

二分 O(n log n)：维护一个 tails 数组，tails[i] = 长度为 i+1 的递增子序列的最小末尾元素，用二分查找更新。

4 代码演示
def lengthOfLIS(nums): n = len(nums) dp = [1] * n for i in range(1, n): for j in range(i): if nums[j] < nums[i]: dp[i] = max(dp[i], dp[j] + 1) return max(dp)
public int lengthOfLIS(int[] nums) { int[] dp = new int[nums.length]; Arrays.fill(dp, 1); for (int i = 1; i < nums.length; i++) { for (int j = 0; j < i; j++) { if (nums[j] < nums[i]) { dp[i] = Math.max(dp[i], dp[j] + 1); } } } return Arrays.stream(dp).max().getAsInt(); }
int lengthOfLIS(vector<int>& nums) { int n = nums.size(); vector<int> dp(n, 1); for (int i = 1; i < n; i++) { for (int j = 0; j < i; j++) { if (nums[j] < nums[i]) { dp[i] = max(dp[i], dp[j] + 1); } } } return *max_element(dp.begin(), dp.end()); }

6 易错要点
dp[i] 初始化为 1

每个元素自身构成长度为 1 的递增子序列，所以 dp[i] 至少为 1。

答案不是 dp[n-1]

LIS 不一定以最后一个元素结尾，答案是 max(dp)，不是 dp[n-1]。

严格递增

条件是 nums[j] < nums[i]，不是 ≤。相等不算递增。

7 扩展思考
  • 二分优化 O(n log n)：维护 tails 数组，用 bisect_left 更新，适合 n 很大的场景
  • 输出具体序列：用 prev 数组记录前驱，回溯还原路径
  • 非递减子序列：条件改为 ≤，二分时用 bisect_right
  • 俄罗斯套娃信封：LeetCode 354，二维 LIS，先排序再套用本题
