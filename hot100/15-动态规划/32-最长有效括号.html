#32 最长有效括号

给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

示例 1
输入：s = "(()"
输出：2
有效括号子串为 "()"，长度为 2
示例 2
输入：s = ")()())"
输出：4
有效括号子串为 "()()"，长度为 4
示例 3
输入：s = ""
输出：0
2 题目拆解
括号字符串 ➜ 只含 ( 和 ) ➜ 最长有效连续子串 ➜ 返回长度

关键约束：

  • 有效括号必须连续，不是子序列
  • 需要找最长的有效子串长度
  • 有效括号可以嵌套和并列
3 解题思路

DP O(n)：定义 dp[i] = 以 s[i] 结尾的最长有效括号长度：

s[i]=')' 且 s[i-1]='(' ➜ dp[i] = dp[i-2]+2
s[i]=')' 且 s[i-1]=')' ➜ 若 s[i-dp[i-1]-1]='(' → dp[i]=dp[i-1]+2+dp[i-dp[i-1]-2]

栈解法：栈底保存最后一个未匹配的 ')' 的索引，遇到 '(' 入栈，遇到 ')' 弹栈后用当前索引减栈顶计算长度。

4 代码演示
def longestValidParentheses(s): n = len(s) dp = [0] * n res = 0 for i in range(1, n): if s[i] == ')': if s[i-1] == '(': # ...() dp[i] = (dp[i-2] if i >= 2 else 0) + 2 elif i - dp[i-1] - 1 >= 0 and s[i-dp[i-1]-1] == '(': dp[i] = dp[i-1] + 2 + (dp[i-dp[i-1]-2] if i-dp[i-1]-2 >= 0 else 0) res = max(res, dp[i]) return res
public int longestValidParentheses(String s) { int n = s.length(); int[] dp = new int[n]; int res = 0; for (int i = 1; i < n; i++) { if (s.charAt(i) == ')') { if (s.charAt(i-1) == '(') { dp[i] = (i >= 2 ? dp[i-2] : 0) + 2; } else if (i - dp[i-1] - 1 >= 0 && s.charAt(i-dp[i-1]-1) == '(') { dp[i] = dp[i-1] + 2 + (i-dp[i-1]-2 >= 0 ? dp[i-dp[i-1]-2] : 0); } } res = Math.max(res, dp[i]); } return res; }
int longestValidParentheses(string s) { int n = s.size(); vector<int> dp(n, 0); int res = 0; for (int i = 1; i < n; i++) { if (s[i] == ')') { if (s[i-1] == '(') { dp[i] = (i >= 2 ? dp[i-2] : 0) + 2; } else if (i-dp[i-1]-1 >= 0 && s[i-dp[i-1]-1] == '(') { dp[i] = dp[i-1] + 2 + (i-dp[i-1]-2 >= 0 ? dp[i-dp[i-1]-2] : 0); } } res = max(res, dp[i]); } return res; }

6 易错要点
dp[i] 只在 s[i]=')' 时更新

以 '(' 结尾不可能形成有效括号，dp[i]=0。只有 ')' 才可能匹配。

两种情况的区分

...()：直接 +2；...))：需要跳过中间有效部分找匹配的 '('。

别忘了加上前面的有效部分

...)) 情况中，匹配到 '(' 后还要加上它前面的 dp 值，即 dp[i-dp[i-1]-2]。

7 扩展思考
  • 栈解法：栈底保存 -1（虚拟边界），遇到 '(' 入栈索引，遇到 ')' 弹栈后用 i-stack.top() 计算长度
  • 双向扫描法 O(n) O(1)：从左到右和从右到左各扫一遍，用计数器代替栈
  • 如果包含其他字符：跳过非括号字符，只处理 '(' 和 ')'
  • 如果要求返回具体子串：记录最大长度时的起始位置，最后截取
