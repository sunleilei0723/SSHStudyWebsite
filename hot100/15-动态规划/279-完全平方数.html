#279 完全平方数

给你一个整数 n，返回和为 n 的完全平方数的最少数量。完全平方数是一个整数，其值等于另一个整数的平方。

示例 1
输入：n = 12
输出：3
12 = 4 + 4 + 4 = 2² + 2² + 2²
示例 2
输入：n = 13
输出：2
13 = 4 + 9 = 2² + 3²
2 题目拆解
目标数 n ➜ 可用：1²,2²,3²... ➜ 最少几个凑出 n ➜ 返回最小数量

关键约束：

  • 每个完全平方数可以重复使用 → 完全背包
  • 求的是最少数量，不是方案数
  • 可用的平方数范围：1², 2², ..., ⌊√n⌋²
3 解题思路

DP O(n√n)：定义 dp[i] = 凑出 i 需要的最少完全平方数个数：

dp[0] = 0 ➜ dp[i] = min(dp[i-j²]) + 1 ➜ 遍历所有 j² ≤ i

对每个 i，尝试所有 j 使得 j² ≤ i，取最小值加 1。

4 代码演示
def numSquares(n): dp = [float('inf')] * (n + 1) dp[0] = 0 for i in range(1, n + 1): j = 1 while j * j <= i: dp[i] = min(dp[i], dp[i - j*j] + 1) j += 1 return dp[n]
public int numSquares(int n) { int[] dp = new int[n + 1]; Arrays.fill(dp, Integer.MAX_VALUE); dp[0] = 0; for (int i = 1; i <= n; i++) { for (int j = 1; j * j <= i; j++) { dp[i] = Math.min(dp[i], dp[i - j*j] + 1); } } return dp[n]; }
int numSquares(int n) { vector<int> dp(n + 1, INT_MAX); dp[0] = 0; for (int i = 1; i <= n; i++) { for (int j = 1; j * j <= i; j++) { dp[i] = min(dp[i], dp[i - j*j] + 1); } } return dp[n]; }

6 易错要点
初始化为无穷大

dp 数组初始值设为 inf，dp[0]=0 是递推起点。不要初始化为 0。

完全背包 vs 0-1背包

完全平方数可以重复使用，这是完全背包问题，内层循环方向与 0-1 背包不同。

数学解法

四平方和定理：任何正整数最多可表示为 4 个完全平方数之和，可用于优化。

7 扩展思考
  • 四平方和定理：答案最多为 4，可用数学方法 O(1) 判断是否为 1/2/4
  • BFS 解法：将问题转化为图的最短路径，从 n BFS 到 0
  • 如果限制每个平方数只能用一次：变为 0-1 背包问题
  • 如果求方案数：将 min 改为 sum，dp 初始化 dp[0]=1
