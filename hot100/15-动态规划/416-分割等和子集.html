#416 分割等和子集

给你一个只包含正整数的非空数组 nums。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

示例 1
输入：nums = [1,5,11,5]
输出：true
可以分割成 [1,5,5] 和 [11]，两子集和都为 11
示例 2
输入：nums = [1,2,3,5]
输出：false
总和为 11，无法分成两个等和子集
2 题目拆解
正整数数组 ➜ 总和为偶数？ ➜ 能否凑出 sum/2 ➜ 0-1背包问题

关键约束：

  • 总和必须为偶数，否则直接返回 false
  • 等价于：能否从数组中选出若干元素，使其和恰好为 sum/2
  • 每个元素只能用一次 → 0-1 背包
3 解题思路

DP O(n × target)：定义 dp[j] = 能否用前 i 个数凑出和 j：

dp[0] = true ➜ dp[j] = dp[j] || dp[j-nums[i]] ➜ 内层逆序遍历

0-1 背包：内层循环必须逆序，保证每个数只用一次。空间优化为一维数组。

4 代码演示
def canPartition(nums): total = sum(nums) if total % 2: return False target = total // 2 dp = [False] * (target + 1) dp[0] = True for num in nums: for j in range(target, num - 1, -1): dp[j] = dp[j] or dp[j - num] return dp[target]
public boolean canPartition(int[] nums) { int total = sum(nums); if (total % 2 != 0) return false; int target = total / 2; boolean[] dp = new boolean[target + 1]; dp[0] = true; for (int num : nums) { for (int j = target; j >= num; j--) { dp[j] = dp[j] || dp[j - num]; } } return dp[target]; }
bool canPartition(vector<int>& nums) { int total = accumulate(nums.begin(), nums.end(), 0); if (total % 2) return false; int target = total / 2; vector<bool> dp(target + 1, false); dp[0] = true; for (int num : nums) { for (int j = target; j >= num; j--) { dp[j] = dp[j] || dp[j - num]; } } return dp[target]; }

6 易错要点
总和为奇数直接 false

如果 sum 为奇数，不可能分成两个等和子集，直接返回 false，这是快速剪枝。

内层逆序

0-1 背包必须逆序遍历 j，否则同一个数会被重复使用。这是与完全背包的关键区别。

dp[0] = true

和为 0 总是可以凑出的（不选任何元素），这是递推基础。
