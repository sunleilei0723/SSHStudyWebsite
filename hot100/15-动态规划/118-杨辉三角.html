#118 杨辉三角

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。在杨辉三角中，每个数是它左上方和右上方的数的和。

示例 1
输入：numRows = 5
输出：[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
示例 2
输入：numRows = 1
输出：[[1]]
2 题目拆解
numRows 行 ➜ 每行首尾为 1 ➜ 中间 = 左上+右上 ➜ 逐行生成

关键约束：

  • 每行第一个和最后一个元素都是 1
  • 中间元素 triangle[i][j] = triangle[i-1][j-1] + triangle[i-1][j]
  • 第 i 行有 i 个元素（从1开始计数）
3 解题思路

逐行构建 O(n²)：从第1行开始，每行基于上一行生成：

第1行 [1] ➜ 第2行 [1,1] ➜ 第i行：首1 + 中间(左上+右上) + 尾1

核心公式：row[j] = prev[j-1] + prev[j]，边界 row[0] = row[-1] = 1

4 代码演示
def generate(numRows): res = [] for i in range(numRows): row = [1] * (i + 1) # 全填1 for j in range(1, i): # 跳过首尾 row[j] = res[i-1][j-1] + res[i-1][j] res.append(row) return res
public List<List<Integer>> generate(int numRows) { List<List<Integer>> res = new ArrayList<>(); for (int i = 0; i < numRows; i++) { List<Integer> row = new ArrayList<>(); for (int j = 0; j <= i; j++) { if (j == 0 || j == i) row.add(1); else row.add(res.get(i-1).get(j-1) + res.get(i-1).get(j)); } res.add(row); } return res; }
vector<vector<int>> generate(int numRows) { vector<vector<int>> res; for (int i = 0; i < numRows; i++) { vector<int> row(i + 1, 1); for (int j = 1; j < i; j++) row[j] = res[i-1][j-1] + res[i-1][j]; res.push_back(row); } return res; }

6 易错要点
行号从0还是1

代码中行索引从 0 开始，第 i 行有 i+1 个元素。注意 range(1, i) 跳过首尾。

边界全为1

每行首尾都是 1，可以先全填 1 再修改中间值，避免边界判断。

numRows=1

特殊情况 numRows=1 时只返回 [[1]]，循环内 j 的范围 [1,0) 为空，不会出错。

7 扩展思考
  • 返回第 k 行：LeetCode 119，只用 O(k) 空间，从右往左更新
  • 组合数公式：杨辉三角第 n 行第 k 个数 = C(n, k)，可用公式直接计算
  • 对称性：每行左右对称，只需计算一半
  • 与二项式定理的关系：第 n 行对应 (a+b)ⁿ 的展开系数
