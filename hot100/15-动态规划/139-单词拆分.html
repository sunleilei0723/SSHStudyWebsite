#139 单词拆分

给你一个字符串 s 和一个字符串列表 wordDict 作为字典。请你判断是否可以利用字典中出现的单词拼接出 s。不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。

示例 1
输入：s = "leetcode", wordDict = ["leet","code"]
输出：true
"leetcode" 可以拆分为 "leet" + "code"
示例 2
输入：s = "applepenapple", wordDict = ["apple","pen"]
输出：true
"applepenapple" = "apple" + "pen" + "apple"
示例 3
输入：s = "catsandog", wordDict = ["cats","dog","sand","and","cat"]
输出：false
2 题目拆解
字符串 s ➜ 字典 wordDict ➜ 能否拆分成字典词 ➜ 返回 true/false

关键约束：

  • 字典中的单词可以重复使用
  • 拆分后的单词必须完全覆盖 s，不能有剩余字符
  • 单词的顺序必须与 s 中的出现顺序一致
3 解题思路

DP O(n²)：定义 dp[i] = s[0:i] 能否被字典拆分：

dp[0] = true ➜ dp[i] = dp[j] && s[j:i]∈dict ➜ 遍历 j ∈ [0, i)

对每个位置 i，检查是否存在 j 使得 dp[j]=true 且 s[j:i] 在字典中。

4 代码演示
def wordBreak(s, wordDict): word_set = set(wordDict) n = len(s) dp = [False] * (n + 1) dp[0] = True for i in range(1, n + 1): for j in range(i): if dp[j] and s[j:i] in word_set: dp[i] = True break return dp[n]
public boolean wordBreak(String s, List<String> wordDict) { Set<String> wordSet = new HashSet<>(wordDict); boolean[] dp = new boolean[s.length() + 1]; dp[0] = true; for (int i = 1; i <= s.length(); i++) { for (int j = 0; j < i; j++) { if (dp[j] && wordSet.contains(s.substring(j, i))) { dp[i] = true; break; } } } return dp[s.length()]; }
bool wordBreak(string s, vector<string>& wordDict) { unordered_set<string> wordSet(wordDict.begin(), wordDict.end()); int n = s.size(); vector<bool> dp(n + 1, false); dp[0] = true; for (int i = 1; i <= n; i++) { for (int j = 0; j < i; j++) { if (dp[j] && wordSet.count(s.substr(j, i - j))) { dp[i] = true; break; } } } return dp[n]; }

6 易错要点
dp[0] = true

空字符串视为可拆分，这是递推的起点。不要忘记初始化。

用 set 加速查找

将 wordDict 转为 HashSet，使 s[j:i] in dict 的查找为 O(1)。

找到就 break

内层循环只要找到一个 j 使 dp[j]=true 且 s[j:i]∈dict，dp[i] 就为 true，可以立即 break。

7 扩展思考
  • 返回所有拆分方案：LeetCode 140，在 DP 基础上回溯构造所有方案
  • Trie 优化：用前缀树代替 set，可以提前终止无效匹配
  • BFS 解法：将位置作为节点，可达位置连边，BFS 判断终点可达性
  • 如果字典很大：可以只检查以 s[j] 开头的字典词，而非枚举所有 j
