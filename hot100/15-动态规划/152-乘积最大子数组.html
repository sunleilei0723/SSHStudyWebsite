#152 乘积最大子数组

给你一个整数数组 nums，请你找出数组中乘积最大的非空连续子数组，并返回该子数组所对应的乘积。子数组是数组的连续子序列。

示例 1
输入：nums = [2,3,-2,4]
输出：6
子数组 [2,3] 的乘积最大，2×3 = 6
示例 2
输入：nums = [-2,0,-1]
输出：0
结果不能为 2，因为 [-2,-1] 不是连续子数组
2 题目拆解
整数数组 ➜ 含负数和0 ➜ 连续子数组 ➜ 最大乘积

关键约束：

  • 子数组必须连续
  • 负数×负数 = 正数，所以需要同时维护最大值和最小值
  • 遇到 0 时乘积归零，需要重新开始
3 解题思路

DP O(n)：同时维护 maxDP[i] 和 minDP[i]：

maxDP[i] = 以 i 结尾的最大乘积 ➜ minDP[i] = 以 i 结尾的最小乘积 ➜ 负数时交换

当 nums[i] < 0 时，交换 maxDP[i-1] 和 minDP[i-1]，因为负数会让最大变最小、最小变最大。

转移：maxDP[i] = max(nums[i], maxDP[i-1]×nums[i])，minDP 同理。

4 代码演示
def maxProduct(nums): max_dp = min_dp = res = nums[0] for i in range(1, len(nums)): if nums[i] < 0: max_dp, min_dp = min_dp, max_dp # 负数交换 max_dp = max(nums[i], max_dp * nums[i]) min_dp = min(nums[i], min_dp * nums[i]) res = max(res, max_dp) return res
public int maxProduct(int[] nums) { int maxDp = nums[0], minDp = nums[0], res = nums[0]; for (int i = 1; i < nums.length; i++) { if (nums[i] < 0) { int t = maxDp; maxDp = minDp; minDp = t; } maxDp = Math.max(nums[i], maxDp * nums[i]); minDp = Math.min(nums[i], minDp * nums[i]); res = Math.max(res, maxDp); } return res; }
int maxProduct(vector<int>& nums) { int maxDp = nums[0], minDp = nums[0], res = nums[0]; for (int i = 1; i < nums.size(); i++) { if (nums[i] < 0) swap(maxDp, minDp); maxDp = max(nums[i], maxDp * nums[i]); minDp = min(nums[i], minDp * nums[i]); res = max(res, maxDp); } return res; }

6 易错要点
必须同时维护 maxDP 和 minDP

负数会让最大乘积变最小，最小乘积变最大。只维护一个会漏掉情况。

负数时先交换再更新

遇到负数时，先交换 maxDP 和 minDP，再用常规公式更新。

与 nums[i] 比较

max(nums[i], maxDP*nums[i])：如果前面乘积不如自身大，就从当前重新开始。

7 扩展思考
  • 如果求最小乘积子数组：同理维护 maxDP 和 minDP，返回 minDP 的全局最小值
  • 如果数组含小数：算法不变，但需注意浮点精度
  • 乘积为正的最长子数组：LeetCode 1567，需要记录正负乘积的长度
  • 前缀积 + 单调栈：另一种思路，但不如双 DP 直观
