#138 随机链表的复制

给定一个带 random 指针的链表，构造一个深拷贝。每个节点有 val、next 和 random 三个属性，random 可以指向任意节点或 null。

2 题目拆解
遍历原链表 ➜ 创建副本节点 ➜ 建立映射关系 ➜ 设置next和random

关键约束：

  • 必须深拷贝，不能引用原链表节点
  • random 指针可能指向尚未创建的节点
  • 需要建立 原节点 → 副本节点 的映射
3 解题思路

哈希表法 O(n) 空间：两次遍历

第1遍：创建副本，存入map ➜ 第2遍：设置next和random ➜ 通过map查找副本节点

拼接法 O(1) 空间：将副本插在原节点后面，最后拆分

4 代码演示
def copyRandomList(head): # 第1遍：创建副本节点 old_to_new = {} cur = head while cur: old_to_new[cur] = Node(cur.val) cur = cur.next # 第2遍：设置next和random cur = head while cur: old_to_new[cur].next = old_to_new.get(cur.next) old_to_new[cur].random = old_to_new.get(cur.random) cur = cur.next return old_to_new.get(head)
public Node copyRandomList(Node head) { // 第1遍：创建副本节点 Map<Node, Node> map = new HashMap<>(); for (Node cur = head; cur != null; cur = cur.next) { map.put(cur, new Node(cur.val)); } // 第2遍：设置next和random for (Node cur = head; cur != null; cur = cur.next) { map.get(cur).next = map.get(cur.next); map.get(cur).random = map.get(cur.random); } return map.get(head); }
Node* copyRandomList(Node* head) { // 第1遍：创建副本节点 unordered_map<Node*, Node*> mp; for (Node* cur = head; cur; cur = cur->next) { mp[cur] = new Node(cur->val); } // 第2遍：设置next和random for (Node* cur = head; cur; cur = cur->next) { mp[cur]->next = mp[cur->next]; mp[cur]->random = mp[cur->random]; } return mp[head]; }

6 易错要点
两次遍历

必须先创建所有副本再设置指针。如果一边创建一边设置 random，目标节点可能还未创建，map.get() 返回 null。

map.get 处理 null

cur.next 或 cur.random 可能为 null，map.get(null) 返回 null，正好对应副本的 null，无需特殊处理。

O(1) 空间解法

拼接法：将副本插在原节点后面，A→A'→B→B'→...，设置完 random 后再拆分。省去哈希表但代码更复杂。

7 扩展思考
  • 拼接法 O(1) 空间：A→A'→B→B'，random 指向 cur.random.next，最后拆分
  • 图的深拷贝：#133 克隆图是类似问题，也用哈希表映射
  • 如果还有 parent 指针：同样用哈希表，多设一个属性即可
  • 序列化方法：先序列化再反序列化，但效率较低
