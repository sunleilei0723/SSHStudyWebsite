#148 排序链表

给定链表的头节点 head，按升序排列并返回排序后的链表。要求 O(n log n) 时间和 O(1) 空间（常数级额外空间）。

示例 1
输入：head = [4,2,1,3]
输出：[1,2,3,4]
2 题目拆解
找中点分割 ➜ 递归排序两半 ➜ 合并有序链表 ➜ 返回排序结果

关键约束：

  • O(n log n) 时间 → 排除插入排序、冒泡排序
  • O(1) 额外空间 → 排除递归（栈空间 O(log n)）和数组辅助
  • 链表不支持随机访问 → 归并排序最适合
3 解题思路

归并排序（自顶向下）：递归分割 + 合并

快慢指针找中点 ➜ 断开成两半 ➜ 递归排序左右 ➜ 合并两个有序链表

自底向上归并：O(1) 空间，用循环代替递归，面试加分项

4 代码演示
def sortList(head): if not head or not head.next: return head # 1. 找中点（快慢指针） slow, fast = head, head.next while fast and fast.next: slow = slow.next fast = fast.next.next # 2. 断开 mid = slow.next slow.next = None # 3. 递归排序 left = sortList(head) right = sortList(mid) # 4. 合并 return merge(left, right) def merge(l1, l2): dummy = cur = ListNode(0) while l1 and l2: if l1.val < l2.val: cur.next, l1 = l1, l1.next else: cur.next, l2 = l2, l2.next cur = cur.next cur.next = l1 or l2 return dummy.next
public ListNode sortList(ListNode head) { if (head == null || head.next == null) return head; // 1. 找中点 ListNode slow = head, fast = head.next; while (fast != null && fast.next != null) { slow = slow.next; fast = fast.next.next; } // 2. 断开 ListNode mid = slow.next; slow.next = null; // 3. 递归排序 ListNode left = sortList(head), right = sortList(mid); // 4. 合并 return merge(left, right); } ListNode merge(ListNode l1, ListNode l2) { ListNode dummy = new ListNode(0), cur = dummy; while (l1 != null && l2 != null) { if (l1.val < l2.val) { cur.next = l1; l1 = l1.next; } else { cur.next = l2; l2 = l2.next; } cur = cur.next; } cur.next = l1 != null ? l1 : l2; return dummy.next; }
ListNode* sortList(ListNode* head) { if (!head || !head->next) return head; // 1. 找中点 ListNode *slow = head, *fast = head->next; while (fast && fast->next) { slow = slow->next; fast = fast->next->next; } // 2. 断开 ListNode* mid = slow->next; slow->next = nullptr; // 3. 递归排序 ListNode *left = sortList(head), *right = sortList(mid); // 4. 合并 return merge(left, right); } ListNode* merge(ListNode* l1, ListNode* l2) { ListNode *dummy = new ListNode(0), *cur = dummy; while (l1 && l2) { if (l1->val < l2->val) { cur->next = l1; l1 = l1->next; } else { cur->next = l2; l2 = l2->next; } cur = cur->next; } cur->next = l1 ? l1 : l2; return dummy->next; }

6 易错要点
断开链表

找到中点后必须 slow.next = None 断开，否则递归会死循环。两半各自独立才能正确排序。

快指针起始位置

找中点时 fast = head.next（不是 head），确保偶数个节点时 slow 偏左，避免死循环。

合并函数

合并两个有序链表是核心子操作，同 #21。用 dummy 节点简化处理，最后别忘了 cur.next = l1 or l2。

7 扩展思考
  • 自底向上归并 O(1) 空间：步长从1开始倍增，每轮合并相邻子链表，无需递归
  • 为什么不用快排：链表快排的 partition 不稳定，最坏 O(n²)；归并始终 O(n log n)
  • 插入排序：#147 对链表做插入排序，O(n²) 但常数小，短链表更快
  • 数组排序 vs 链表排序：数组可随机访问适合快排，链表适合归并
