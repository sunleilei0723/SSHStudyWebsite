#02 两数相加

两个非空链表表示两个非负整数，每位数字按逆序存储。将两数相加，返回表示和的链表。如 l1=[2,4,3], l2=[5,6,4] → 342+465=807 → 返回 [7,0,8]。

示例 1
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
342 + 465 = 807
2 题目拆解
两个逆序链表 ➜ 逐位相加 ➜ 处理进位 ➜ 返回结果链表

关键约束：

  • 数字按逆序存储 → 个位在头节点，天然对齐
  • 每位相加可能产生进位 carry（0或1）
  • 两个链表长度可能不同，短链表用0补齐
  • 最高位可能还有额外进位，需新增节点
3 解题思路

迭代法：同时遍历两个链表，逐位相加，维护进位

取 l1.val + l2.val + carry ➜ 新节点 = sum % 10 ➜ carry = sum / 10 ➜ 移动 l1, l2

递归法：递归处理下一位，返回值带上进位，代码更简洁

4 代码演示
def addTwoNumbers(l1, l2): dummy = cur = ListNode(0) carry = 0 while l1 or l2 or carry: val1 = l1.val if l1 else 0 val2 = l2.val if l2 else 0 total = val1 + val2 + carry carry = total // 10 cur.next = ListNode(total % 10) cur = cur.next l1 = l1.next if l1 else None l2 = l2.next if l2 else None return dummy.next
public ListNode addTwoNumbers(ListNode l1, ListNode l2) { ListNode dummy = new ListNode(0), cur = dummy; int carry = 0; while (l1 != null || l2 != null || carry > 0) { int val1 = (l1 != null) ? l1.val : 0; int val2 = (l2 != null) ? l2.val : 0; int total = val1 + val2 + carry; carry = total / 10; cur.next = new ListNode(total % 10); cur = cur.next; l1 = (l1 != null) ? l1.next : null; l2 = (l2 != null) ? l2.next : null; } return dummy.next; }
ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) { ListNode *dummy = new ListNode(0), *cur = dummy; int carry = 0; while (l1 || l2 || carry) { int val1 = l1 ? l1->val : 0; int val2 = l2 ? l2->val : 0; int total = val1 + val2 + carry; carry = total / 10; cur->next = new ListNode(total % 10); cur = cur->next; l1 = l1 ? l1->next : nullptr; l2 = l2 ? l2->next : nullptr; } return dummy->next; }

6 易错要点
最后还有进位

循环条件必须包含 carry > 0，否则 [9,9] + [1] 会丢失最后的进位节点 1。

链表长度不同

短链表遍历完后用 0 补齐，不能直接停止。如 [9,9,9] + [1] 需要继续处理。

dummy节点

使用虚拟头节点 dummy 统一处理，避免对头节点的特殊判断。最后返回 dummy.next。

7 扩展思考
  • 如果数字是正序存储：需要先反转链表或用栈处理，见 #445 两数相加 II
  • 递归写法：递归到末尾，返回时带上进位，代码更简洁但可能栈溢出
  • 多个链表相加：扩展为 k 个链表，用队列逐位处理
  • 大数加法：链表相加本质是大数加法，字符串加法同理
