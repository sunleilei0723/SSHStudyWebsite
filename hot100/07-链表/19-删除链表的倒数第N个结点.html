#19 删除链表的倒数第N个结点

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。要求使用一趟扫描实现。

示例 1
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
2题目拆解
单链表 ➜ 删除倒数第N个 ➜ 快慢指针间隔N ➜ slow.next = slow.next.next

关键约束：

  • 一趟扫描完成，不能先遍历算长度再删
  • 需要找到倒数第 N 个的前驱节点才能删除
  • 用 dummy 节点处理删除头节点的边界情况
3解题思路

快慢指针法 O(n) O(1)

fast 先走 N 步，然后 fast 和 slow 同步前进。当 fast 到达末尾时，slow 恰好在倒数第 N 个节点的前驱位置。

fast 先走 N 步 ➜ fast 和 slow 同步前进 ➜ fast 到末尾 ➜ 删除 slow.next
4代码演示
def removeNthFromEnd(head, n): dummy = ListNode(0, head) fast = slow = dummy for _ in range(n): # fast 先走 n 步 fast = fast.next while fast.next: # 同步前进 fast = fast.next slow = slow.next slow.next = slow.next.next # 删除 return dummy.next
public ListNode removeNthFromEnd(ListNode head, int n) { ListNode dummy = new ListNode(0, head); ListNode fast = dummy, slow = dummy; for (int i = 0; i < n; i++) fast = fast.next; while (fast.next != null) { fast = fast.next; slow = slow.next; } slow.next = slow.next.next; return dummy.next; }
ListNode* removeNthFromEnd(ListNode* head, int n) { ListNode dummy(0); dummy.next = head; ListNode *fast = &dummy, *slow = &dummy; for (int i = 0; i < n; i++) fast = fast->next; while (fast->next) { fast = fast->next; slow = slow->next; } slow->next = slow->next->next; return dummy.next; }

6易错要点
dummy 节点

必须用 dummy 节点，否则删除头节点时 slow 无法定位前驱（头节点没有前驱）。

fast 先走 N 步

fast 从 dummy 出发走 N 步，不是从 head 出发。这样 slow 最终停在待删节点的前驱位置。

循环条件

同步前进的条件是 fast.next != null，不是 fast != null。fast 停在最后一个节点时 slow 刚好到位。

7扩展思考
  • 两次遍历法：第一次算长度 L，第二次走 L-N 步到前驱，两次扫描
  • 删除中间节点：237 题，只给待删节点，用值覆盖法 node.val = node.next.val; node.next = node.next.next
  • 删除所有指定值节点：203 题，同样用 dummy 简化头节点处理
  • N 大于链表长度：题目保证 N 有效，实际中需加判断
