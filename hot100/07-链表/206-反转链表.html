#206 反转链表

给你单链表的头节点 head，请你反转链表，并返回反转后的链表。

2 题目拆解
单链表 ➜ 反转所有指针 ➜ 三指针迭代 ➜ 返回新头

关键约束：

  • 链表可以迭代或递归反转
  • 核心操作：让每个节点的 next 指向前一个节点
  • 需要 三个指针 prev / curr / next 协作
3 解题思路

迭代法 O(n) O(1)

用三个指针 prev、curr、next：先保存 curr.next，再将 curr.next 指向 prev，然后 prev 和 curr 同时前进一步。

保存 next = curr.next ➜ curr.next = prev ➜ prev = curr ➜ curr = next

递归法 O(n) O(n)

递归到链表末尾，从后往前反转：head.next.next = head，head.next = null。

4 代码演示
def reverseList(head): prev = None curr = head while curr: nxt = curr.next # 保存下一个 curr.next = prev # 反转指针 prev = curr # prev 前进 curr = nxt # curr 前进 return prev # prev 就是新头
public ListNode reverseList(ListNode head) { ListNode prev = null, curr = head; while (curr != null) { ListNode nxt = curr.next; curr.next = prev; prev = curr; curr = nxt; } return prev; }
ListNode* reverseList(ListNode* head) { ListNode *prev = nullptr, *curr = head; while (curr) { ListNode *nxt = curr->next; curr->next = prev; prev = curr; curr = nxt; } return prev; }

6 易错要点
先保存 next

修改 curr.next 之前必须先保存 curr.next，否则修改后无法前进到下一个节点。

prev 初始为 null

prev 初始为 null 不是 head，因为反转后原头节点的 next 应指向 null。

返回 prev 不是 curr

循环结束时 curr = null，prev 恰好指向新头节点。返回 curr 会得到 null。

7 扩展思考
  • 递归写法：reverseList(head.next) 返回子链表新头，然后 head.next.next = head
  • 反转前 N 个节点：记录第 N+1 个节点，反转后接到尾部
  • 反转区间 [m, n]：定位 m-1 节点，反转 m 到 n，再接回
  • K 个一组翻转：25 题，迭代 + 反转子函数的组合
