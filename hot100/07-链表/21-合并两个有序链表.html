#21 合并两个有序链表

将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

2题目拆解
两个有序链表 ➜ 双指针比较 ➜ 取较小者接入 ➜ 拼接剩余

关键约束：

  • 两个链表都是升序的
  • 需要原地拼接，不创建新节点
  • 用 哑节点(dummy) 简化头节点处理
3解题思路

迭代法 O(n+m) O(1)

用双指针 p1、p2 分别遍历两个链表，每次取较小者接入结果链表，对应指针前进一步。一方用尽后直接拼接另一方剩余部分。

p1.val ≤ p2.val? ➜ 取较小者接入 ➜ 对应指针前进 ➜ 拼接剩余
4代码演示
def mergeTwoLists(l1, l2): dummy = curr = ListNode(0) while l1 and l2: if l1.val <= l2.val: curr.next = l1; l1 = l1.next else: curr.next = l2; l2 = l2.next curr = curr.next curr.next = l1 or l2 # 拼接剩余 return dummy.next
public ListNode mergeTwoLists(ListNode l1, ListNode l2) { ListNode dummy = new ListNode(0), curr = dummy; while (l1 != null && l2 != null) { if (l1.val <= l2.val) { curr.next = l1; l1 = l1.next; } else { curr.next = l2; l2 = l2.next; } curr = curr.next; } curr.next = l1 != null ? l1 : l2; return dummy.next; }
ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) { ListNode dummy(0), *curr = &dummy; while (l1 && l2) { if (l1->val <= l2->val) { curr->next = l1; l1 = l1->next; } else { curr->next = l2; l2 = l2->next; } curr = curr->next; } curr->next = l1 ? l1 : l2; return dummy.next; }

6易错要点
哑节点技巧

使用 dummy 节点避免对头节点的特殊处理，最后返回 dummy.next。

拼接剩余

一方遍历完后，另一方剩余部分直接整体接入，不需要逐个遍历。

等值取 l1

当 l1.val == l2.val 时取 l1 或 l2 都行，保持稳定性建议统一取 l1。

7扩展思考
  • 递归写法：比较头节点，较小者的 next 指向递归合并结果
  • 合并 K 个有序链表：23 题，用分治或最小堆
  • 原地合并：不使用 dummy，需要额外处理头节点判断
  • 降序合并：修改比较方向即可
