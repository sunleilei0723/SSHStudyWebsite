#234 回文链表

给定一个单链表的头节点 head，判断该链表是否为回文链表。要求 O(n) 时间和 O(1) 空间。

2 题目拆解
输入链表 ➜ 快慢指针找中点 ➜ 反转后半段 ➜ 前后对比

关键约束：

  • 回文 = 正读反读相同，如 1→2→2→1
  • 要求 O(1) 空间 → 不能用栈/数组存全部节点
  • 快慢指针找中点 + 反转后半段是最优解
3 解题思路

栈法 O(n) 空间：遍历链表压栈，再遍历与栈顶比较 → 空间 O(n)，不满足要求

快慢指针 + 反转 O(1) 空间：

快指针走2步，慢指针走1步 ➜ 慢指针到中点 ➜ 反转后半段 ➜ 双指针从两端对比
4 代码演示
def isPalindrome(head): # 1. 快慢指针找中点 slow = fast = head while fast and fast.next: slow = slow.next fast = fast.next.next # 2. 反转后半段 prev = None while slow: nxt = slow.next slow.next = prev prev = slow slow = nxt # 3. 从两端对比 left, right = head, prev while right: if left.val != right.val: return False left = left.next right = right.next return True
public boolean isPalindrome(ListNode head) { // 1. 快慢指针找中点 ListNode slow = head, fast = head; while (fast != null && fast.next != null) { slow = slow.next; fast = fast.next.next; } // 2. 反转后半段 ListNode prev = null; while (slow != null) { ListNode nxt = slow.next; slow.next = prev; prev = slow; slow = nxt; } // 3. 从两端对比 ListNode left = head, right = prev; while (right != null) { if (left.val != right.val) return false; left = left.next; right = right.next; } return true; }
bool isPalindrome(ListNode* head) { // 1. 快慢指针找中点 ListNode *slow = head, *fast = head; while (fast && fast->next) { slow = slow->next; fast = fast->next->next; } // 2. 反转后半段 ListNode *prev = nullptr; while (slow) { ListNode* nxt = slow->next; slow->next = prev; prev = slow; slow = nxt; } // 3. 从两端对比 ListNode *left = head, *right = prev; while (right) { if (left->val != right->val) return false; left = left->next; right = right->next; } return true; }

6 易错要点
快慢指针边界

当节点数为偶数时，slow 停在右半段起点；为奇数时停在正中间。反转后半段时奇数中间节点不参与比较，不影响结果。

反转后断开

反转后半段时，前半段最后一个节点的 next 仍指向原中点，但比较时 right 先到 null，所以不影响。不需要显式断开。

恢复链表

如果要求不修改原链表，比较完后需要再次反转恢复。面试中应主动提及这一点。

7 扩展思考
  • 如果允许 O(n) 空间：用栈或数组存所有值，再双端对比，代码更简单
  • 如果要求恢复原链表：比较完再反转后半段接回去
  • 递归解法：利用递归栈天然逆序遍历，但空间 O(n)，且可能栈溢出
  • 进阶：判断回文数组 vs 回文链表的核心区别在于链表不能随机访问
