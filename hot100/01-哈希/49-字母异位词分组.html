#49 字母异位词分组

给你一个字符串数组 strs，请你将字母异位词组合在一起。字母异位词是由相同字母重新排列形成的不同字符串。可以按任意顺序返回结果列表。

示例 1
输入：strs = ["eat","tea","tan","ate","nat","bat"]
输出：[["bat"],["nat","tan"],["ate","eat","tea"]]
示例 2
输入：strs = [""]
输出：[[""]]
2 题目拆解
字符串数组 ➜ 排序/计数规范化 ➜ 按键分组 ➜ 返回分组列表

关键约束：

  • 字符串仅含小写字母，排序和计数都可行
  • 需要一个统一的键来标识同一组字母异位词
  • 输出顺序不限，分组内顺序也不限
3 解题思路

排序作键 O(n·k log k)：对每个字符串排序，排序后相同的归为一组。如 "eat"→"aet"，"tea"→"aet"，键相同。

计数作键 O(n·k)：统计每个字母出现次数，将计数数组转为字符串作键。如 "eat"→"#1#0#0#0#1#0...#1#0"。

遍历每个字符串 ➜ 排序/计数生成 key ➜ 哈希表按 key 分组 ➜ 返回所有分组
4 代码演示
from collections import defaultdict def groupAnagrams(strs): groups = defaultdict(list) # key → 字符串列表 for s in strs: key = ''.join(sorted(s)) # 排序作键 groups[key].append(s) # 按键分组 return list(groups.values())
public List<List<String>> groupAnagrams(String[] strs) { Map<String, List<String>> map = new HashMap<>(); for (String s : strs) { char[] chars = s.toCharArray(); Arrays.sort(chars); String key = new String(chars); // 排序作键 map.computeIfAbsent(key, k -> new ArrayList<>()).add(s); } return new ArrayList<>(map.values()); }
vector<vector<string>> groupAnagrams(vector<string>& strs) { unordered_map<string, vector<string>> groups; for (auto& s : strs) { string key = s; sort(key.begin(), key.end()); // 排序作键 groups[key].push_back(s); } vector<vector<string>> res; for (auto& [k, v] : groups) res.push_back(v); return res; }

6 易错要点
键的选择

排序作键时，必须对排序后生成新字符串，不能直接用原串。Python 用 ''.join(sorted(s))，Java 用 new String(chars)。

Python defaultdict

使用 defaultdict(list) 避免每次判断键是否存在。普通 dict 需 if key not in d，容易遗漏导致 KeyError。

计数作键用元组

Python 中列表不可哈希，计数法必须将计数列表转为 tuple 才能作为 dict 的键，否则报 TypeError。

7 扩展思考
  • 如果字符集扩展到 Unicode：排序法仍适用，计数法需要更大数组不再高效
  • 质数乘法作键：为每个字母分配唯一质数(a=2,b=3,c=5...)，异位词乘积相同。键是整数哈希更快，但长字符串可能溢出
  • 模式提炼："规范化 + 哈希分组" — 找到函数 f 使 f(x)=f(y) 当且仅当 x、y 同组，可推广到等价类划分
  • 如果要求组内也排序：最终对每个子列表排序即可，不影响整体复杂度
