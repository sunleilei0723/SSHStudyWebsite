#128 最长连续序列

给定一个未排序的整数数组 nums，找出数字连续的最长序列的长度。要求算法时间复杂度为 O(n)。序列不要求元素在原数组中连续排列。

示例 1
输入：nums = [100,4,200,1,3,2]
输出：4
最长数字连续序列是 [1, 2, 3, 4]，长度为 4
示例 2
输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9
2 题目拆解
无序数组 ➜ 去重入集合 ➜ 找序列起点 ➜ 向右延伸 ➜ 记录最长

关键约束：

  • 必须 O(n) 时间复杂度 → 不能排序
  • 数组可能含重复元素 → 先去重
  • 序列不要求在原数组中连续排列
3 解题思路

排序法 O(n log n)：排序后遍历统计连续段 → 不满足 O(n) 要求，淘汰

哈希集合 O(n)：将所有元素放入集合，只从序列起点（num-1 不在集合中）开始向右延伸，避免重复计算：

转为集合去重 ➜ 遍历：num-1 不在集合？ ➜ 是起点→向右延伸 ➜ 更新最长长度
4 代码演示
def longestConsecutive(nums): num_set = set(nums) # 去重 + O(1)查找 longest = 0 for num in num_set: if num - 1 not in num_set: # 只从起点开始 current = num streak = 1 while current + 1 in num_set: current += 1 streak += 1 longest = max(longest, streak) return longest
public int longestConsecutive(int[] nums) { Set<Integer> set = new HashSet<>(); for (int n : nums) set.add(n); int longest = 0; for (int num : set) { if (!set.contains(num - 1)) { // 只从起点开始 int cur = num, streak = 1; while (set.contains(cur + 1)) { cur++; streak++; } longest = Math.max(longest, streak); } } return longest; }
int longestConsecutive(vector<int>& nums) { unordered_set<int> s(nums.begin(), nums.end()); int longest = 0; for (int num : s) { if (!s.count(num - 1)) { // 只从起点开始 int cur = num, streak = 1; while (s.count(cur + 1)) { cur++; streak++; } longest = max(longest, streak); } } return longest; }

6 易错要点
必须判断起点

如果不判断 num-1 not in set，每个元素都延伸，时间退化为 O(n²)。判断起点是保证 O(n) 的关键。

先去重

数组可能含重复元素，必须先转为集合去重。否则重复元素会导致判断错误。

遍历集合非数组

应遍历 num_set 而非 nums。遍历原数组时重复元素被多次处理，效率降低。

7 扩展思考
  • 并查集解法：对每个 num，若 num+1 也在集合中则合并，最终最大连通分量即为答案。路径压缩后接近 O(n)
  • 如果要求返回序列本身：延伸时记录起点和终点，最后重建 [start..end] 即可
  • 模式提炼："只从起点出发" — 识别序列/区间起点，只从起点处理，避免重复。可推广到合并区间、统计岛屿数量
  • 动态规划 + 哈希表：用哈希表记录每个数所在连续序列长度，遇到 num 时查 num-1 和 num+1 的长度合并更新，一次遍历完成
