#994 腐烂的橘子

在给定的 m x n 网格中，每个单元格可以有三个值之一：值 0 代表空单元格，值 1 代表新鲜橘子，值 2 代表腐烂的橘子。每分钟，腐烂的橘子会向上下左右四个方向传播腐烂。返回直到单元格中没有新鲜橘子为止所必须经过的最小分钟数。如果不可能，返回 -1。

示例 1
输入：grid = [[2,1,1],[1,1,0],[0,1,1]]
输出：4
第0分钟左上角腐烂，4分钟后所有橘子腐烂
示例 2
输入：grid = [[2,1,1],[0,1,1],[1,0,1]]
输出：-1
左下角橘子被0隔开，永远不会腐烂
2 题目拆解
初始腐烂橘子 ➜ 每分钟向四周扩散 ➜ BFS 逐轮扩散 ➜ 统计分钟数

关键约束：

  • 多源 BFS：所有初始腐烂橘子同时开始扩散
  • 如果存在新鲜橘子永远无法被感染，返回 -1
  • 每轮扩散 = 1 分钟，需要逐层BFS
3 解题思路

多源 BFS O(m×n)：先将所有腐烂橘子入队，然后逐层 BFS 扩散。每处理完一层，分钟数 +1。最后检查是否还有新鲜橘子。

所有腐烂橘子入队 ➜ 逐层BFS扩散 ➜ 感染相邻新鲜橘子 ➜ 检查剩余新鲜橘子

def orangesRotting(grid): m, n = len(grid), len(grid[0]) q = deque() fresh = 0 for i in range(m): for j in range(n): if grid[i][j] == 2: q.append((i,j)) elif grid[i][j] == 1: fresh += 1 mins = 0 while q and fresh: for _ in range(len(q)): # 逐层 x,y = q.popleft() for dx,dy in [(1,0),(-1,0),(0,1),(0,-1)]: nx,ny = x+dx, y+dy if 0<=nx<m and 0<=ny<n and grid[nx][ny]==1: grid[nx][ny] = 2 fresh -= 1 q.append((nx,ny)) mins += 1 return mins if fresh == 0 else -1

6 易错要点
多源BFS

必须将所有初始腐烂橘子同时入队，不是逐个处理。否则时间计算错误。

逐层计时

每处理完一整层（for _ in range(len(q))）才 +1 分钟，不是每感染一个就 +1。

不可能的情况

BFS 结束后如果还有 fresh > 0，说明有橘子被空格隔开，返回 -1。

7 扩展思考
  • 01矩阵：求每个格子到最近0的距离，同样是多源BFS
  • 地图分析：求离陆地最远的海洋格，多源BFS从所有陆地出发
  • 墙壁与门：每个房间到最近门的距离，多源BFS
  • 如果腐烂有延迟：不同橘子腐烂速度不同，需要优先队列（Dijkstra）
