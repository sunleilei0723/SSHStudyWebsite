#5 最长回文子串

给定一个字符串 s，找到 s 中最长的回文子串。如果存在多个最长回文子串，返回第一个。

示例 1
输入：s = "babad"
输出："bab"
"aba" 同样是合法答案
示例 2
输入：s = "cbbd"
输出："bb"
2 题目拆解
字符串 s ➜ 找回文 ➜ 最长子串 ➜ 返回子串

关键约束：

  • 子串（连续），不是子序列
  • 回文 = 正读反读一样，如 "aba"、"bb"
  • 长度 ≥ 1 的子串一定是回文
3 解题思路

中心扩展法 O(n²)：以每个字符（和每两个字符之间）为中心，向两边扩展：

选中心 ➜ 左右扩展 ➜ 不等则停 ➜ 记录最长

DP 法 O(n²)：dp[i][j] 表示 s[i..j] 是否回文，dp[i][j] = dp[i+1][j-1] && s[i]==s[j]

Manacher O(n)：最优解但较复杂，面试一般掌握中心扩展即可

4 代码演示
def longestPalindrome(s): def expand(l, r): while l >= 0 and r < len(s) and s[l] == s[r]: l -= 1; r += 1 return s[l+1:r] # 返回回文子串 res = s[0] for i in range(len(s)): odd = expand(i, i) # 奇数长度 even = expand(i, i+1) # 偶数长度 res = max(res, odd, even, key=len) return res
public String longestPalindrome(String s) { String res = s.substring(0, 1); for (int i = 0; i < s.length(); i++) { String odd = expand(s, i, i); String even = expand(s, i, i+1); if (odd.length() > res.length()) res = odd; if (even.length() > res.length()) res = even; } return res; } String expand(String s, int l, int r) { while (l >= 0 && r < s.length() && s.charAt(l) == s.charAt(r)) { l--; r++; } return s.substring(l+1, r); }
string longestPalindrome(string s) { int n = s.size(), start = 0, maxLen = 1; auto expand = [&](int l, int r) { while (l >= 0 && r < n && s[l] == s[r]) { l--; r++; } if (r-l-1 > maxLen) { start = l+1; maxLen = r-l-1; } }; for (int i = 0; i < n; i++) { expand(i, i); // 奇数 expand(i, i+1); // 偶数 } return s.substr(start, maxLen); }

6 易错要点
奇偶中心

必须同时考虑奇数长度（单字符中心）和偶数长度（双字符中心）的回文。只考虑一种会漏解。

边界返回

扩展结束后，回文范围是 [l+1, r-1]（因为退出时 s[l]!=s[r]），注意 +1 和 -1。

初始值

至少有一个字符是回文，所以初始 res = s[0]，长度至少为 1。

7 扩展思考
  • Manacher 算法：O(n) 时间，通过插入分隔符统一奇偶，用已知回文加速
  • 回文子串个数：遍历每个中心，统计每个中心能扩展出的回文数
  • 最长回文子序列：不要求连续，用区间 DP，dp[i][j] 表示最长回文子序列长度
  • 分割回文串：LeetCode 131，回溯 + DP 判断回文
