#72 编辑距离

给定两个单词 word1 和 word2，返回将 word1 转换成 word2 所使用的最少操作数。你可以对一个单词进行三种操作：插入一个字符、删除一个字符、替换一个字符。

示例 1
输入：word1 = "horse", word2 = "ros"
输出：3
horse → rorse (替换 h→r) → rorse → rose (删除 r) → rose → ros (删除 e)
示例 2
输入：word1 = "intention", word2 = "execution"
输出：5
2 题目拆解
word1 ➜ 三种操作 ➜ 最少操作数 ➜ 变 word2

三种操作：

  • 插入：在 word1 中插入一个字符
  • 删除：删除 word1 中的一个字符
  • 替换：替换 word1 中的一个字符
3 解题思路

状态定义：dp[i][j] = word1[0..i-1] 变成 word2[0..j-1] 的最少操作数

转移方程：

若 w1[i-1]==w2[j-1] ➜ dp[i][j] = dp[i-1][j-1]
若不等 ➜ min(插入, 删除, 替换) + 1

三种操作对应：

  • 插入：dp[i][j-1] + 1（word2 多匹配一个）
  • 删除：dp[i-1][j] + 1（word1 删一个）
  • 替换：dp[i-1][j-1] + 1（换一个字符）

边界：dp[i][0]=i（删 i 次），dp[0][j]=j（插 j 次）

4 代码演示
def minDistance(word1, word2): m, n = len(word1), len(word2) dp = [[0]*(n+1) for _ in range(m+1)] for i in range(m+1): dp[i][0] = i for j in range(n+1): dp[0][j] = j for i in range(1, m+1): for j in range(1, n+1): if word1[i-1] == word2[j-1]: dp[i][j] = dp[i-1][j-1] else: dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1 return dp[m][n]
public int minDistance(String word1, String word2) { int m = word1.length(), n = word2.length(); int[][] dp = new int[m+1][n+1]; for (int i = 0; i <= m; i++) dp[i][0] = i; for (int j = 0; j <= n; j++) dp[0][j] = j; for (int i = 1; i <= m; i++) for (int j = 1; j <= n; j++) if (word1.charAt(i-1) == word2.charAt(j-1)) dp[i][j] = dp[i-1][j-1]; else dp[i][j] = Math.min(dp[i-1][j], Math.min(dp[i][j-1], dp[i-1][j-1])) + 1; return dp[m][n]; }
int minDistance(string word1, string word2) { int m = word1.size(), n = word2.size(); vector<int> dp(n+1); for (int j = 0; j <= n; j++) dp[j] = j; for (int i = 1; i <= m; i++) { int prev = dp[0]; dp[0] = i; for (int j = 1; j <= n; j++) { int tmp = dp[j]; if (word1[i-1] == word2[j-1]) dp[j] = prev; else dp[j] = min({dp[j], dp[j-1], prev}) + 1; prev = tmp; } } return dp[n]; }

6 易错要点
边界初始化

dp[i][0]=i（删 i 次），dp[0][j]=j（插 j 次）。忘记初始化会导致错误。

相等时直接继承

字符相等时 dp[i][j]=dp[i-1][j-1]，不需要 +1，因为无需操作。

三种操作别混淆

插入对应 dp[i][j-1]+1，删除对应 dp[i-1][j]+1，替换对应 dp[i-1][j-1]+1。方向别搞反。

7 扩展思考
  • 与 LCS 的关系：编辑距离是 LCS 的推广，LCS 只允许删除，编辑距离允许插入、删除、替换
  • 不同操作代价：如果插入/删除/替换代价不同，只需把 +1 改成对应代价
  • 只允许插入和删除：答案 = m + n - 2*LCS
  • 回文编辑距离：将字符串变成回文的最少操作数
