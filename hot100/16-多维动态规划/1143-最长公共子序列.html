#1143 最长公共子序列

给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。如果不存在公共子序列，返回 0。

示例 1
输入：text1 = "abcde", text2 = "ace"
输出：3
最长公共子序列是 "ace" ，长度为 3
示例 2
输入：text1 = "abc", text2 = "abc"
输出：3
2 题目拆解
两个字符串 ➜ 找公共子序列 ➜ 最长长度 ➜ 返回长度

关键约束：

  • 子序列不要求连续，但保持相对顺序
  • 空串的 LCS 长度为 0
  • 两个字符串长度可达 1000
3 解题思路

状态定义：dp[i][j] = text1[0..i-1] 和 text2[0..j-1] 的 LCS 长度

转移方程：

若 s₁[i-1]==s₂[j-1] ➜ dp[i][j] = dp[i-1][j-1] + 1
若 s₁[i-1]≠s₂[j-1] ➜ dp[i][j] = max(dp[i-1][j], dp[i][j-1])

边界：dp[0][j]=0, dp[i][0]=0（空串与任何串的 LCS 为 0）

时间 O(mn)，空间 O(mn)

4 代码演示
def longestCommonSubsequence(text1, text2): m, n = len(text1), len(text2) dp = [[0]*(n+1) for _ in range(m+1)] for i in range(1, m+1): for j in range(1, n+1): if text1[i-1] == text2[j-1]: dp[i][j] = dp[i-1][j-1] + 1 else: dp[i][j] = max(dp[i-1][j], dp[i][j-1]) return dp[m][n]
public int longestCommonSubsequence(String text1, String text2) { int m = text1.length(), n = text2.length(); int[][] dp = new int[m+1][n+1]; for (int i = 1; i <= m; i++) for (int j = 1; j <= n; j++) if (text1.charAt(i-1) == text2.charAt(j-1)) dp[i][j] = dp[i-1][j-1] + 1; else dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]); return dp[m][n]; }
int longestCommonSubsequence(string text1, string text2) { int m = text1.size(), n = text2.size(); vector<int> dp(n+1, 0); for (int i = 1; i <= m; i++) { int prev = 0; for (int j = 1; j <= n; j++) { int tmp = dp[j]; if (text1[i-1] == text2[j-1]) dp[j] = prev + 1; else dp[j] = max(dp[j], dp[j-1]); prev = tmp; } } return dp[n]; }

6 易错要点
子序列 vs 子串

子序列不要求连续，子串要求连续。LCS 是子序列问题，不要和最长公共子串混淆。

dp 下标偏移

dp[i][j] 对应 text1[0..i-1] 和 text2[0..j-1]，比较的是 text1[i-1] 和 text2[j-1]，注意 -1。

不等时取 max

字符不等时，dp[i][j] = max(dp[i-1][j], dp[i][j-1])，不是 dp[i-1][j-1]。

7 扩展思考
  • 输出 LCS 字符串：从 dp[m][n] 反向追踪，相等则加入结果并走对角线，不等走较大值方向
  • 最长公共子串：要求连续，dp[i][j] 只在相等时 = dp[i-1][j-1]+1，不等时归零
  • 多个字符串的 LCS：扩展到三维 dp，时间复杂度指数增长
  • 编辑距离：LCS 的自然扩展，允许插入、删除、替换操作
