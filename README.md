# SSHStudyWebsite

LeetCode Hot 100 手绘视频风教学卡片集，按技术分类组织的交互式算法学习卡片。

## 在线预览

直接用浏览器打开 `index.html`（根目录入口）或 `hot100/index.html`（题目导航页）即可浏览。

## 项目特色

- **手绘视频风格** — 纸白底+点阵网格背景、厚黑边框+偏移阴影、五色系统（紫/绿/粉/黄/蓝），Baloo 2 + Nunito + Roboto Mono 字体组合
- **8 章节完整内容** — 题目 → 题目拆解 → 解题思路 → 代码演示 → 题解过程 → 易错要点 → 扩展思考 → 相关题目
- **三语言代码** — Python / Java / C++ 一键切换，语法高亮 + 行号 + 关键行标记 + Copy 按钮
- **交互式逐步演示** — ◀ ▶ AUTO 控制，柱状图/矩阵网格/链表节点/树结构等图形化可视化，直观展示算法执行过程
- **单文件自包含** — 每张卡片一个 HTML 文件，无外部依赖（除 Google Fonts CDN），可直接离线使用

## 卡片内容结构

每张卡片包含 8 个章节：

1. **题目** — 题号、标题、难度标签、算法标签、题目描述
2. **题目拆解** — 流程图式拆解 + 关键约束列表
3. **解题思路** — 暴力法对比 + 最优解思路 + 流程图
4. **代码演示** — Python/Java/C++ 三语言切换 + 语法高亮 + 关键行标记
5. **题解过程** — 交互式逐步演示，图形化可视化算法执行
6. **易错要点** — 三栏笔记卡片，提炼常见陷阱
7. **扩展思考** — 变体问题、进阶思路
8. **相关题目** — 同类型题目链接

## 技术栈

- 纯 HTML/CSS/JS，单文件自包含
- Google Fonts CDN（Baloo 2, Nunito, Roboto Mono）
- CSS 变量系统 + 响应式布局
- prefers-reduced-motion 适配
